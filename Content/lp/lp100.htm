Label Printing Functions

Reset Eltron Printer

 Last Revised: 07/31/13

Reset Eltron Printer is used to reset the Eltron printer to the default settings, i.e. ones that are known to be compatible with Dynamo. Dynamo communicates to Eltron and Zebra brand bar code printer using the Eltron Printer Control language. If a bar code printer's language or other settings have been changed, Dynamo may have trouble communicating with the printer. When this application is used, these disruptions are eliminated by resetting the printer.
