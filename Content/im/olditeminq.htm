If the item is an alternate item, it will display the total alternate items available. The FET Sell, FET Cost, AVG Warehouse, STD Warehouse, Cost, and Costing Unit will also be displayed if the operator has a security level of 5 or greater. At this point, a window will display with all of the options that are available through Item inquiry.

This option is made available automatically if the inventory contains serialized items.

The operator is then asked to enter a Status Code

  1. Available for sale
  2. Committed on sales orders
  3. Committed on transfer orders
  4. Sold
  5. Rented
  6. Cost information

or any combination of the six. Next the operator is asked to enter a valid warehouse code. If all warehouse codes are desired then the operator may just hit Enter. If there are any serialized items for the given status code(s) and warehouse code(s) then the following information is displayed for each item:

  • Line Number, ID Number
  • Status Code, Color
  • Warehouse, New / Used Indicator
  • Serial Number & Description, Selling Price
