Product Class Listing

 Last Revised: 10/04/13

This program displays all of the product classes in Dynamo. The resulting list contains the category, the category description, the product class, the product class description, the rent product class (where applicable), and the used product class (where applicable). The operator only must choose a printer for the listing. Product Classes are created in Product Class Maintenance.
