Warehouse Location Label Printing

 Last Revised: 02/06/14

Warehouse Location Label Printing is used to print labels for a range of items or a range of item locations. This application is useful for preparing a warehouse for inventory if you specify a range of item locations. An operator can also print slightly different looking warehouse location labels from the ICR menu at Warehouse Location Labels.

Sample Warehouse Location Label printouts.

 

Item Search Options Invoked with F5
Exclude Discontinued Items with no QOH Enable to exclude items that have no Quantity on Hand.
Exclude Items if Warehouse Qty Avail=0 Enable to exclude items unavailable in the warehouse.
Exclude Items if Company Qty Avail=0 Enable to exclude items unavailable at the company.
