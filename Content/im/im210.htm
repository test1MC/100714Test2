Item Maintenance Utility Functions

Alternate Item Analyzer

 Last Revised: 02/06/14

The Alternate Item Analyzer is used to print a display of any issues within the alternate item files. The application can determine whether an alternate item is missing or if an alternate item has another alternate item. It also identifies whether or not an alternate item points to itself as an alternate item. The item number and the alternate item number are given so that the issue can be located and corrected.
