Item Listing

 Last Revised: 02/19/14

This program lists all of the items entered into Dynamo based on an operator-specified range, sequence, and format. The operator must also specify whether or not to include items that have no barcode.

The possible formats are as follows: 

Item Listing Options
1. All Items
  • item number
  • item description
  • package quantities
  • weight
  • barcode (where applicable)
2. Vendor Part Numbers
  • item number
  • item description
  • vendor number and alpha code
  • vendor part number
  • cost
  • unit of measure
  • as of date
3. Items with an Alternate Item
  • item number
  • item description
  • alternate item number
  • Alternate Item Usage Code
  • transfer quantity
4. Bar Codes
  • item number
  • item description
  • whether or not item is tracked in inventory
  • quantity
  • barcode

5. Items with Track Inventory=N
  • item number
  • item description
  • whether or not item is tracked in inventory

6. Items with Track Inventory=N and inventory

  • item number
  • item description
  • whether or not item is tracked in inventory
  • quantity on hand
7. Discounted Items with inventory
  • item number
  • item description
  • discount date
  • quantity on hand of the item
