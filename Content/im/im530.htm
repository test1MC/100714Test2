Generate ABC Values

 Last Revised: 12/10/13

Generate ABC Values is used to create letter codes that represent frequency of sale for items. These frequency values are based on times sold within the past 12 months. The available codes are A, B, C, D, and blank. Each symbol represents a different percentage of the company's item sales.

ABC Code Values
A Top 20% (most frequently sold items)
B Next 60%
C Last 20% (least frequently sold items)
D Items with a QOH but no sales
Blank Items with no QOH or sales

ABC classifications are calculated automatically at the end of each month, but the operator can also manually run Generate ABC Values at any time.
