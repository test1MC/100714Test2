Warehouse Location Maintenance

 Last Revised: 01/07/14

Warehouse Location Maintenance is used to add or delete item locations. The operator can quickly find an item in the warehouse based on the listed locations. The numbers signifying the location of items may indicate the building, the section, the row, or a particular shelf in that row; it all depends on a company's numbering system. Multiple locations are possible.

Warehouse Location Maintenance View
Warehouse Type in the name of the desired warehouse.
Loc'n Type in available locations in the warehouse.

If the operator enters a warehouse location number that does not follow the format of the previously entered warehouse locations, the application will not allow the new one to be entered.
