Vendor Label Printing

 Last Revised: 08/30/13

Vendor Label Printing will print vendors in alphabetical order using the primary address.  

The labels used are 8.5 x 11" sheets with 3 labels across and 10 labels down. There is a 1/2" margin at the top and bottom of each page. Excellware provides the correct laser labels for this application. Call Excellware at (440) 866-6893 for current pricing on these labels.

Labels can be printed in all CAPS or in Upper/Lower case. You will be able to print one page at a time or the first page alone, followed by the remaining pages (using the F2 key).
