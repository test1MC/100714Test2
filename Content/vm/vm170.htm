Change Vendor Number

 Last Revised: 01/08/14

Change Vendor Number allows the operator to change a vendor's number within Dynamo. The application is rarely necessary, but there are two situations in which it is sometimes used.

  1. If a vendor was mistakenly entered into the system twice, this program will allow the operator to merge the information associated with each vendor number into one of the vendor numbers.
  2. If a permanent vendor was mistakenly entered with a single character vendor code, the number can be corrected here. Single character vendor codes are only used for one-time vendors.

The operator chooses a "change from" vendor number and a "change to" vendor number.
