Vendor Address Correction

 Last Revised: 07/12/13

Vendor Address Correction allows the batch correction of vendor addresses within Dynamo. The application verifies the addresses by comparing them to the post office listing. It can fix the spelling of street names, normalize the addresses (abbreviates street, road, drive, etc.), and verify the Zip+4 code. The program can also discover that a number in the address is out of range (it doesn't exist on that particular street), but at that point you must manually correct the number.

After the program has automatically corrected any errors, it will prompt you to correct them manually if necessary. The vendor addresses from this program print so you can view any errors within your addresses. Once you have viewed the list, you can manually update any addresses in Dynamo.
