Vendor Maintenance & Listings

Vendor Listing

 Last Revised: 01/17/14

A Vendor Listing can be printed or displayed to the screen. 

Vendor Listing Formats
Name and Addresses
  • Vendor Alpha Sort Name
  • Vendor Number
  • Vendor Name
  • Vendor Address including city, state, zip code
  • Vendor Telephone Number
Contacts
  • Vendor Alpha Sort Name
  • Vendor Number
  • Vendor Name
  • Contact Name
  • Contact Person's Department
  • Contact's Phone #s
  • Contact's E-mail Address
  • Other Name
Accounts Payable Information

Indicate all the payment methods to include on the listing:

  • blank
  • (S) Separate Check
  • (T) Electronic Fund Transfer
  • (W) Automated Withdrawal

This listing is a summary of A/P detail found in each vendor record.  It includes:

  • Vendor Alpha Sort Name
  • Vendor Number
  • Vendor Name
  • Telephone Number and Alternate Number
  • Location Code
  • Default G/L Account Number
  • Default Terms Code
  • 1099 Code indicating Form Type and Description
  • Taxpayer ID Number/Social Security Number
  • Payment Cycle Code
  • Hold Status (Yes or No)
  • Payment Method (blank=normal check is printed, S=Separate Check,  T=Electronic Fund Transfer, or W=Automated Withdrawal)
  • Customer Number (by which this vendor identifies you)
Purchase Order Processing Information This listing is a summary of POP detail found in each vendor record.   It includes:
  • Alpha Sort Name
  • Vendor Number
  • Vendor Name
  • Buyer Code indicating Buyer's Name
  • Terms Code
  • Taxable Status
  • Freight On Board at Shipping Point or Destination
  • Freight Code
  • Weight, Amount, and Volume Information
Vendors with Freight/Duty or Rebate%

This listing displays information about vendors with freight/duty or rebate% found in each vendor record. It includes:

  • Vendor#
  • Vendor Name
  • Buyer
  • Freight/Duty%
  • Rebate%

The Payment Methods field only needs to be filled in if you have chosen the Accounts Payable Information format. The available payment methods are blank (normal), S (separate check), T (transfer of funds), and W (automated withdrawal).
