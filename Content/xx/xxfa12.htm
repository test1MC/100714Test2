Fixed Asset Category Maintenance

 Last Revised: 11/12/14

Fixed Asset Category Maintenance allows the operator to create and edit category codes for fixed assets. Categories could relate to computer equipment, vehicles, shop equipment, communications equipment, office furniture, warehouse equipment, rental equipment, etc.

Fixed Asset Category Maintenance View
Category Code You can maintain an existing fixed asset category by typing its code or selecting it after pressing F2. To create a new fixed asset category, type an unused code.
Description Maintain the description for the fixed asset category.
GLA# Asset Specify the general ledger account number where the asset will be posted to.
GLA# Accumulated Depreciation Specify the general ledger account where the accumulated depreciation on the fixed asset will be posted to.
GLA# Depreciation Expense Specify the general ledger account where the depreciation expense will be posted to.

Any of the fields can be changed by typing their corresponding number or deleted by typing D.
