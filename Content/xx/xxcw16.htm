Dynascan User Maintenance

 Last Revised: 01/09/14

Dynascan User Maintenance allows the operator to edit who can operate Dynascan on a terminal scanner. Login IDs are created here. If you do not have Dynascan, you can explore its uses in the Dynascan Menu Simulator on the SMU menu.

Dynascan User Maintenance View
Warehouse Enter in the warehouse associated with this hand-held device operator.
Operator ID Indicate the operator ID for the specified warehouse.
