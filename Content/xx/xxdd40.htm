System Management Utility Functions

Dynascan Menu Maintenance

 Last Revised: 01/28/14

Dynascan is a collection of Dynamo applications that run in a hand held computer with an integrated bar code scanner. Dynascan applications can be used to receive product, put-away product, pick products, pack products, record physical inventory, etc.

This program is used to add menu selections to the Dynascan menu that is displayed on the hand-held scanner.

Dynascan Menu Maintenance Primary View
Menu Code Type in or choose the menu code.
Title Type in or change the title of the menu.
Dynascan Menu Maintenance Secondary View
Menu Heading Use this option to choose the menu code.
Menu Detail

Use this to add or remove programs/menus on your hand-held scanner.

* This column will contain an asterisk on the line you are currently editing.
Type This column signifies if you are adding a Program or a Menu.
Prog/Menu This column displays the name of the program or menu.
Description This column contains the description of the program or menu.
