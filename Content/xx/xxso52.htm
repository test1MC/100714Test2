Cash Drawer Paid Out Code Maintenance

 Last Revised: 11/12/14

Cash Drawer Paid Out Code Maintenance is used to define the codes used for money taken out of the cash drawer for a service/other purpose. For instance, a window washing service may be used for your company. If cash is paid to them out of the cash drawer, that code needs to be defined here. This application also defines the General Ledger account the transaction is posted to.

Cash Drawer Paid Out Code Maintenance View
Cash Paid Out Code Use this field to create a code for a service that requires cash paid out.
Description Use this field to describe the code.
G/L Account# Specify a general ledger to which this cash paid out code will post to.
