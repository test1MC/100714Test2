Dynaweb Management Software

Dynaweb Login Maintenance

 Last Revised: 01/09/14

This application is used to maintain Dynaweb login information.

 

Some Intranet installations disable the Dynaweb login in which case this application is not required. Some of the fields below are not applicable for intranet-only installations.

Option Description
  The Login ID is an 8 character field.  It can also be the same as the Unix login for employee operators
1 The operator's name
2 The Dynaweb password, 5 characters minimum
3 Indicates if the operator is an employee, customer, sales rep, or other operator.  It is used for certain Dynaweb statistical web activity reports, and to limit access to customer specific data.
  Customer number and name is used when the Dynaweb login is for a customer.
  Salesman number and name is used when the Dynaweb login is for a salesman.
4 Email address of Dynaweb operator.  This is used should the operator forget his password.  Dynaweb can email the password to this address.
5 Company Code - only used in multi-company installations where a common cgi script is used with multiple Dynamo companies.  Normally the cgi script varies by company code.
6 The login can be manually or automatically disabled after not being used for a number of days, as specified in Dynaweb Control Maintenance.
  The most recent, and previous login date and time are displayed.
