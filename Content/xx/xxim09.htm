item maintenance utility functions

Change Item# Entry

 Last Revised: 12/18/13

Change Item# Entry is used to change the number that identifies a particular item in Dynamo, as these numbers cannot be changed in Item Maintenance. After using this application, a listing of all item number changes can be viewed in Change Item# Listing. To apply these changes, the operator must use Change Item# Update.

This application may be used if item numbers have changed due to switching suppliers. The operator can change the number instead of creating a new number, and then the sales history will stay linked to the item.
