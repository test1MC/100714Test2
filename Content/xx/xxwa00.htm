Dynaweb Management Software

Dynaweb Control Maintenance

 Last Revised: 01/09/14

This application is used with Dynaweb to provide basic configuration options.

Option# Description
1 Optional URL to point to a logo file that displayed on the login page and each Dynaweb menu page.
2 Administrator's email address which is used by Dynaweb applications to notify in case of an error.
3 Optional feature to disable a Dynaweb login that has not been used in the days specified.
4 Optional parent URL used on the main menu as a link.
5 Name of the person to show on the main login page as a contact to request a login or password maintenance.
6 The phone number and extension of the login administrator
7 The email address of the login administrator.
