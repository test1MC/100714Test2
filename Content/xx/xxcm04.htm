Customer Type Maintenance

 Last Revised: 08/06/13

Use this program to create and edit customer types. For example:

  • DR: Dealer, Auto, Truck, Off-Road
  • FL: Fleet
  • OS: Office Supply
  • RL: Retail
  • **: All Customers
Customer Type Maintenance View
Customer Type Type in an existing code to edit an existing customer type or enter an unused code to create a new customer type.
Description Maintain the associated description for the customer type indicated.

Customer types can be used for easy grouping on promotional pricing or marketing to specific types of customers.
