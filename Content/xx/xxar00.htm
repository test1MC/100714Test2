A/R Parameter Maintenance

 Last Revised: 01/27/15

A/R Parameter Maintenance is used to specify whether or not addresses will print on statements as well as define finance charges. The finance charge grace period, charge rate, and minimum charge can all be specified within this application. Enter a number to edit the corresponding field.

A/R Parameter Maintenance Primary View
Print Addresses on Statements Type "Y" or "N" to indicate whether or not addresses will print on statements.
Finance Charge Grace Period Days Indicate how many days past due must occur before a finance charge will apply (i.e., the grace period).
Finance Charge Rate Indicate the finance charge rate.
Minimum Finance Charge Indicate the lowest rate at which finance charges will apply. For example, if you set this field as 5.00, any finance charges below that rate will not be applied to a customer's account.
