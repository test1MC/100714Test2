Prospect maintenance and listings

Prospect Maintenance

 Last Revised: 11/12/14

Prospect Maintenance is used to create new and edit previously entered prospects.

Prospect Maintenance View
Prospect# To edit an existing prospect, type their prospect number or search for it using the options at the bottom of the screen. To create a new prospect, press space bar to use the next available number or press F2 to manually assign the number.
Name Maintain the name of the prospect.
Address Line #1, #2 Enter the address of the prospect. When possible, it is best to fit the entire street address on one line, including suite or apartment numbers. However, if there are two lines of address present, such as if an "Attention" line needs to be included, type the street address in the second line. Dynamo reads the second line of address first.
City, State/Country, Zip Code, County Indicate the prospect's zip code. The city, state, and county will be automatically populated.
Alpha Sort Name Decide how this customer will be alphabetized. This will be used to help you locate customers when searching. For instance, you may want to use the main noun in a company name or the last name of a customer who is an individual.
Telephone# Include the prospect's phone number.
Alternate Telephone# Include the prospect's alternate phone number. This is typically a fax number, an 800 number, or a toll-free number.
Web Address Indicate the prospect's website.
Source Code Indicate a source code; source codes can be created in Prospect Source Code Maintenance. A source indicates how you obtained or found the prospect.
Customer Type Indicate a customer type; customer types can be maintained in Customer Type Maintenance.
SIC# Indicate the prospect's Standard Industrial Classification number. The SIC code is a 4-digit business category code assigned by the U.S. government. Click here to go to the SIC Code website that identifies all SIC codes.
Employees List any known employee names.
Approximate Sales Indicate an estimated amount of sales made by this prospect in order to indicate the approximate size of the company.
Selection Code[1][2][3][4] Selection codes are used to group prospects together for emailing or physical mailing purposes.

Contacts View
Name Type in the name of a contact associated with this prospect.
Department/Title Enter the title or department of this contact, such as Sales Manager or Sales.
Telephone#

Enter the contact's phone numbers, such as a main work phone number and/ or a cell phone number.
Email Address Enter the contact's email address.
Dear Name Indicate how you'd refer to this contact in correspondence, i.e. the name he or she would prefer to be called.
Selection Code This field allows you to define certain groups within a particular company. Contacts with the same Selection Code would be part of the same group.

Notepad View
Display/Modify Existing Notepad Edit any previously entered information for this prospect.
Create New Notepad for [Today's Date] Create a new note for this prospect.
