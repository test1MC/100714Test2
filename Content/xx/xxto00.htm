Transfer Order Parameter Maintenance

 Last Revised: 01/09/14

Transfer Order Parameter Maintenance is used to edit default information for transfer orders. You can change the next transfer order number, the preferred carrier, the service, the billing method, the backorder type, and the security level to update.

Transfer Order Parameter Maintenance View
Next Transfer Order# This option allows the operator to designate the number for the next order made in transfer order processing.
Preferred Carrier This option allows the operator to specify the preferred carrier from a list. To choose a carrier not on the list, select "OTH" for Other.
Service

When selecting OTH, this menu displays these choices:

  • Best Way
  • Will Call
  • Customer Pickup
  • Our Truck
Billing Method

This option allows the operator to choose the billing method from the following:

  • Prepay & Add
  • Prepay (Paid by Vendor)
  • No Charge
  • Freight Collect
Backorder Type

This option allows the operator to choose the backorder type from the following:

  • Default to Zero
  • No Backorders
  • Default to Balance
Security Level to Update Use this option to designate the security level necessary to update transfer orders. To manage security levels, go to Operator Information Maintenance. Security levels are defined in Menu Control Maintenance.
