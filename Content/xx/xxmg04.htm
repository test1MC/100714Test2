Magento Category Inquiry

 Last Revised: 03/25/16
   

Magento Categories (and subcategories) are used to organize products into groups so that customers can easily browse and find what they are looking for. If used, these categories are set in the Magento Admin Panel. In order to use this menu, the option to Use Product Class as Mag Category in the Magento Integration menu must be set to No(N). Otherwise, Dynamo will have no information to display here since it would be using the Dynamo Product Class in place of a Magento Category.

 

Entering the Magento Category ID will return the name of the attribute set. The F2 hotkey can also be used to select from a list of available Magento categories. This menu may be used to verify that a product category recently added to Magento is currently reflected in Dynamo. If not, try choosing Update Magento Attribute Sets under the Magento Integration menu as this also updates Magento categories.
