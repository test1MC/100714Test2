Commission Code Maintenance

 Last Revised: 01/09/14

Commission Code Maintenance is used to create and edit commission codes used for salespersons.

A commission code setup may look like this, for example:

Commission Code Maintenance Primary View
Commission Code Enter an existing code to maintain its description and structure. Type an unused code to create a new one.
Description Indicate how the commission code is used or who it is used for with a description.
Structure
GM% Indicate the gross margin percent to be gained using the current commission code.
Commission% Indicate the commission percent for the current commission code.
Sales/Margin Indicate whether the commission is paid based on sales or margin.

Type "D" to delete the current record.

To view a list of all commission codes created, use Commission Code Listing.
