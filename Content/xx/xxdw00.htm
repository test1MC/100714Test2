Dynaweb Control Maintenance

 Last Revised: 01/09/14

Dynaweb Control Maintenance is used to specify information related to company Dynaweb accounts. The administrator listed here is the person typically contacted about server issues.

Dynaweb Control Maintenance View
Logo URL Insert a link to your company's logo here.
Admin Email Address Type the email address associated with the administrator.
Parent URL Indicate the URL of your company's website.
Login Admin Name Type the name of the login administrator.
Login Admin Phone Type the phone number of the login administrator
Login Admin Email Type the e-mail of the login administrator.
Ribbon Color Bkg Type the hexadecimal code of the ribbon color background.
Ribbon Color Font Type the hexadecimal code of the ribbon color font.
Disable Logins Type "Y" to disable logins or "N" not to disable them.
Login Message Enter in a message to be seen upon login.
