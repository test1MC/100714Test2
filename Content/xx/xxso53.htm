accounts receivable Utility Functions

Cash Drawer Maintenance

 Last Revised: 11/12/14

This application is used to maintain the information for all cash drawers used in Dynamo.

Cash Drawer Maintenance View
Cash Drawer# To edit information about an existing cash drawer, type its number or press F2 to display a list of all entered cash drawers. To create a new cash drawer, type an unused number.
Description Type a description related to the current cash drawer (such as location).
Branch Choose the associated branch from the list by pressing F2.
Bank Code Choose the associated bank code from the list by pressing F2. Bank codes can be created in Bank Code Maintenance.
Beginning Balance Type in the beginning balance of the current cash drawer.
