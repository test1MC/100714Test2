Item Label Design Maintenance

 Last Revised: 11/12/14

This application allows the operator to create labels for items that do not have them. The operator can also design how labels will be printed from Dynamo.

Item Label Maintenance View
Item Label Select an existing label design by typing its name or press F2 to select it from a list. To create a new label name, just type an unused title.
Description Describe what this item design is used for.
Use Label Configuration If you've already created an item label and want to reuse settings from that label, use this option to select which design you want to follow.
Labels Across Indicate how many labels across will print.
Orientation

The labels can be oriented in any of the following ways:

  • Normal
  • Rotated 90 degrees clockwise
  • Inverted 180 degrees
  • Bottom Up 270 degrees clockwise
Columns Indicate how many columns of labels are on the printed sheet.
Rows Indicate how many rows of labels are on the printed sheet.
