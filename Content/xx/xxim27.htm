Product Class Maintenance

 Last Revised: 03/30/16

This application allows the operator to create and edit product classes. Each product class must be assigned a three digit code and given a description. Classes can also be given product class rental descriptions and product class used sales descriptions.

Product Class Maintenance Primary View
Product Class The Product Class identifier needs to be three characters. It can be comprised of any combination of numbers, uppercase letters, lowercase letters, or special characters. The first character in the identifier should signify the Product Category. For example, if the Product Class is 802, the related Product Category should be 8.
Description Enter a description of the product class. For example, in the product category of vehicles, possible product class descriptions might be cars, vans, SUVs, trucks, etc.
Product Class - Rentals Specify the product class that is associated with the general ledger account used for rental items. This general ledger account can be specified on the GLU G/L Interface Maintenance. This field is not required.
Product Class - Used Sales Specify the product class that is associated with the general ledger account used for used item sales. This general ledger account can be specified on the GLU G/L Interface Maintenance. This field is not required.
Attribute Set Attribute sets are used when integrating Dynamo with Magento, an eCommerce software. Since attribute sets are created within the Magento software, no attribute sets will appear here if not already created in Magento and updated by Dynamo. If already created and updated, choose which attribute set will be associated with all items bearing the current product class.

Selling Prices by Product Class View
Customer# Indicate if the pricing will apply to a specific customer. If it does not, you may leave this field blank. You can apply different special pricing for different customers.
Pricing Start Date Indicate a start date for the pricing.
Promo Pricing Ends Indicate an end date for the pricing or leave this field blank to extend the pricing indefinitely.
Pricing Method

Indicate the pricing method:

  • G Gross Margin%
  • C Multiplier times Cost
  • D Discount% off List Price
  • M Multiplier times List Price
Factor Enter the desired gross margin percent, the cost multiplier, the discount percent, or the list price multiplier.
