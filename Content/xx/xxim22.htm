Item Label Functions

Item Label Info Maintenance

 Last Revised: 01/09/14

Item Label Info Maintenance is used to create and edit the information on a label. Labels are not necessary for all companies, but if your company manufactures products, this can be a useful tool.

Item Label Info Maintenance Primary View
Item# Select an item whose label you want to create.
Item Label Choose a design for the label. The designs are specified in Item Label Design Maintenance.
Size Where applicable, indicate the size of the item. This number will display to the left of the description of the item.
Text[1] Type text that will appear on the first line of the label.
Text[2] Type text that will appear on the second line of the label.
Text[3] Type text that will appear on the third line of the label.
