sales order processing Utility Functions

BOL Classification Maintenance

 Last Revised: 01/09/14

Use this program to create freight classes for shipping. You can assign these codes to sales orders so that a shipper knows what type of item it is based on weight, density, fragility, or hazard. Carriers typically do not need to know the exact type of item (or item description) that they will be carrying, just what type of item, so they know how to ship it. Specifying the class code here allows the code to appear on all bills of lading so it does not need to be retyped every time.

BOL Classification Maintenance Primary View
ClassCode Enter an existing code or create a new code for the bill of lading class.
Description Maintain the description of the bill of lading class.

Type "D" to delete the record.
