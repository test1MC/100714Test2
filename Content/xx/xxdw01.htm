Dynaweb Management Software

Dynaweb Login Maintenance

 Last Revised: 01/09/14

Use this program to create login IDs for www.excellware.com. You can also edit pre-existing login IDs and their related information. Dynaweb stores information from Dynamo so that your customers can view data related to their orders.

Dynaweb Login Maintenance Primary View
Login ID Indicate the login ID for Dynaweb.
Name Enter the name associated with the login ID.
Email Address Enter the e-mail address associated with the login ID. Generally, the email address is now used as the login ID for Dynaweb.
User Type

Enter the User type:

  • Employee
  • Salesman
Development Prefix This field specifies the development prefix necessary to test any applications before they're submitted live.
