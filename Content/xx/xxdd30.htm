Dynaweb Menu Maintenance

 Last Revised: 01/28/14

This application is used to create a simple menu of options for Dynaweb operators.  The menu is presented in a centered single column HTML table with hyperlinks for menu selections.  The colors and fonts used by the menu are controlled by the cascading style sheet.

Each menu is assigned a three character menu code.  Each menu can contain up to 255 menu selections.

Menu Selection Type Description Parameters
H - Heading Used to create a heading line for a group of applications that follow Heading Text
P - Program Used to invoke a Dynaweb program or application Program Name
M - Menu Used to link a sub-menu Sub-menu code
L - Link Used to link any URL Descriptive text and hyperlink
