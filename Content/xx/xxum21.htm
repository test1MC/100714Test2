Warehouse Location Maintenance by Item

 Last Revised: 02/05/15

Warehouse Location Maintenance by Item allows the operator to specify where an item is located using a company-specific code style.

Warehouse Location Maintenance by Item
Item# Type in the number for an item you want to specify locations for.
Warehouse Indicate the warehouse where the item is located.
Locations

Type in where this item is located using your company's location mask system. The operator can use the Insert key to enter numbers or the Delete key to remove them.

Location number systems are set up in Warehouse Information Maintenance in the Location Mask field on the CWP menu.
