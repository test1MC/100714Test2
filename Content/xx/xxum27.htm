Inquiry Functions

Product Class Inquiry

 Last Revised: 02/11/06

Product Class Inquiry displays information about each product class.

Entry of the product class number is required; however, pressing F2 will display all product classes in numerical order. After entering the product class number, the following screen is displayed:



The Product Class Inquiry Screen

The only option for this program is to print a hard copy. Answer 'Y' to print a copy, 'N' or 'Enter' if a hard copy is not desired. To exit this program hit F4.
