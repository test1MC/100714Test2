Prospect Source Code Maintenance

 Last Revised: 01/09/14

This application allows the operator to create new and edit pre-existing prospect source codes, which are two-digit codes that describe the source from which a prospect's name, address, and other contact information was obtained. For instance, if the information was obtained at a trade show, the source code could be "TS" and the description would be the date and name of the trade show. The information could also come from a list service that could be identified in this application.
