Shipping Carrier & Service Maintenance

 Last Revised: 03/27/14

Shipping Carrier & Service Maintenance is used to indicate information about how packages should be carried.

Shipping Carrier & Service Maintenance Carrier Info View
Carrier Specify a code for the carrier. Typically, it would be an abbreviation of the carrier, such as FDX for FedEx and UPS for United Parcel Service.
Name Indicate the full name of the carrier.
Std Carrier Alpha Code Enter an alpha code for the carrier to be used for sorting purposes.
Display Sequence Indicate the carrier's priority order number (the lower the number the higher it shows up on a list of carrier choices).
Shipper's Account# Specify the account number of the shipper.
Use on Sales Orders Enable this option if this carrier is used on Sales Orders.
Use on Purchase Orders Enable this option if this carrier is used on Purchase Orders.
Bill Shipper Text Some carriers have specific terms for shipper, receiver, third party, and consignee billing text. Specify the proper bill shipper text here.
Bill Receiver Text Some carriers have specific terms for shipper, receiver, third party, and consignee billing text. Specify the proper bill receiver text here.
Bill Third Party Text Some carriers have specific terms for shipper, receiver, third party, and consignee billing text. Specify the proper bill third party text here.
Consignee Billing Text Some carriers have specific terms for shipper, receiver, third party, and consignee billing text. Specify the proper consignee billing text here.
Web Tracking Link [1][2][3] Enter the URL for how packages can be tracked.

Standard File Maintenance will display a list of carriers to choose from.

Selecting an existing record from the list allows the operator to search for a particular carrier or display all of them by hitting the spacebar.

Max Lbs Enter the maximum package weight in pounds that can be delivered by this service.
Max Ozs Enter the maximum package weight in ounces that can be delivered by this service.
SCC Create a shortcut code for this service.
SOP Enable this option if this service is used in Sales Order Processing.
POP Enable this option if this service is used in Purchase Order Processing.
WEB Enable this option if this service is used on Dynaweb. The operator can also specify "W" for "Warn as expensive."
