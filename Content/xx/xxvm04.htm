Vendor Type Maintenance

 Last Revised: 01/09/14

Vendor Type Maintenance is used to define Vendor Type Codes.  

These codes are optionally assigned to a vendor record in order to group vendors by type (e.g. utilities, resale companies, etc.) on the Vendor Purchases Report found on the APR menu.  

The Vendor Type Maintenance Screen

Vendor Type Code Examples
I Inventory
F Freight
U Utilities
O Other
