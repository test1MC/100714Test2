Magento Attribute Set Inquiry

 Last Revised: 03/25/16
   

Attribute sets are created in Magento and are comprised of a series of specifications or features that are common to a particular type of product. As an example, for computers, the attribute set might contain CPU, RAM, GPU, power supply, USB inputs, etc. Magento will display this information to online customers so that they can compare these attributes among different products when shopping.

Entering the attribute set's ID will return the name of the attribute set. The F2 hotkey can also be used to select from a list of available attribute set IDs. This menu may be used to verify that an attribute set recently added to Magento is reflecting in Dynamo. If not, try choosing Update Magento Attribute Sets under the Magento Integration menu. As this is an inquiry-only menu, changes to attribute sets can not be made here. Any changes to an attribute set itself must be made in the Magento Admin Panel. To change the attribute set associated with a Dynamo product class, please refer to Product Class Maintenance.
