Vendor Returns Processing

 Last Revised: 01/15/15

Vendor Returns Processing is used to return items for credit or replacement. When the vendor return is processed, the inventory value is credited and the account specified in the GLU interface is debited. If the return item is replaced, it can be added into inventory using inventory adjustment type VRR. If a credit memo is issued, it can be entered through the A/P Invoice Data Entry.

Vendor Info

Vendor Info View
Vendor#, Name Input the vendor number or a few letters of the vendor name to populate these fields.
Address Line 1 & 2, City, State/Country, Zip Code These fields will automatically populate with the address on file in Vendor Maintenance, but they can be changed if necessary.
Alpha Sort Name The alphabetization code will automatically populate as specified in Vendor Maintenance for this vendor.
Contact Enter a vendor contact name here, if applicable.
Telephone# This field will automatically populate with the telephone number on file in Vendor Maintenance, but it can be changed if necessary.
Fax# This field will automatically populate with the fax number on file in Vendor Maintenance, but it can be changed if necessary.
Email Address If applicable, supply an email address for the vendor contact.
Shipping Info

Shipping Info View
Warehouse Indicate which warehouse from which you are issuing a return.
Preferred Carrier If a particular carrier is preferred, it can be specified here.
Service If a particular service is preferred, it can be specified here.
Billing Method

Select the billing method for this return:

  • Prepay & Add
  • Prepay (Paid by Vendor)
  • No Charge
  • Freight Collect
Bill Shipping to Acct# If an account is to be billed for shipping, indicate that number here.
FOB Point

You can select Shipping Point or Destination.

Shipping Point indicates the buyer pays shipping cost and takes responsibility for the goods when they leave the seller’s premises. Destination indicates that the seller will pay shipping costs and remain responsible for the goods until the buyer takes possession.

Heading Info

Heading Info
Reference Enter a reference for what was included on the return.
Date New This field is populated automatically and indicates when this vendor return was created.
Date Returned Indicate the date on which the items were returned.
RMA# If you have been given a return merchandise authorization number, input that number in this field.
Internal Notepad If desired, a note that can only be seen in Vendor Returns Processing can be typed here.
External Notepad If desired, a note that will display on the printed vendor return can be typed here.
Total Weight, Total Cube Based on weights and volumes specified in Item Maintenance, these fields will automatically populate with the sum of the weights and volumes of the items on the return.
