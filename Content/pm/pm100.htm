Prospect Inquiry

 Last Revised: 01/07/14

The Prospect Inquiry program allows the operator to search for a prospect within Dynamo.

You can search for the prospect by typing its prospect#, the alpha sort name, the contact name/last name, the phone number, or the zip code.

Prospect Inquiry Primary View
Contact Names View, edit, or change the contact people for the prospect company.
Prospect Information View, edit, or change the codes and contact information for the prospect.
Prospect Notepad View existing notepad information on this prospect or create a new one to include any extra information.
Display Web Site The prospect's website will open in a web browser. The automatic launching of the browser requires use of FacetWin or PowerTerm terminal emulators.
Display Map Google Maps will open in a web browser showing the location of the prospect's company. The automatic launching of the browser requires use of FacetWin or PowerTerm terminal emulators.
