Prospect maintenance & listings

Prospect Listings, Labels, Export

 Last Revised: 07/31/13

This program allows the operator to display a list of operator-designated prospect ranges for marketing or contact purposes.

Formats
Listings

Use this option to list all of the prospects the operator specified.

Name & Address Listing This option displays the alpha code, the prospect number & name, the address, and the telephone number with no contact person.
Contact Listing This option displays the alpha code, the prospect number & name, the address, and the telephone number, as well as the contact person's name and contact information.
Labels

Use this option to print labels to be pressed onto mail.

Exclude Contact Name The labels printed will not contact the contact person's name above the address.
First Contact Name The labels printed will display the first listed contact person's name above the address.
Each Contact Name The labels printed will display all contact names above the address.
Export Use this option to export a spreadsheet of prospects into other programs.

 

Note: to print labels, you need 8.5" x 11" paper with three labels across and ten labels down.
