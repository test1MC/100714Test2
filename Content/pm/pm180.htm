Prospect Import

 Last Revised: 07/29/13

Prospect Import is used to quickly enter contact information and addresses into Dynamo for marketing activities. For instance, if the operator has purchased a list of contacts from a company, he/she can import it into Dynamo. The contacts must be saved in a tab delimited *.txt file and copied to a folder on the Dynamo server. The operator must map the columns from the original document to match their associated fields in the Prospect Import file. This application reads the spreadsheet and incorporates them into Dynamo.

Options
1. Maintain Import Parameters At each field, enter in the column in which that particular piece of information appears in the spreadsheet file.
2. Print Matched Prospect Listing

Choose to sort the file by row or by prospect# using the numbers indicated at the bottom of the screen.

3. Import Prospects

Finally, import the prospect information.
