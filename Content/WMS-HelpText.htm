Help Text

Web Management Software

It is often desirable to have help text that describes, in more detail, the elements on a page, or the steps to be followed. The Excellware Web Management Software includes standards for:

  • where the help text should be located,
  • what the help text should be named,
  • the method of linking the help text to a style sheet, and
  • the method of displaying the help text.
Location

The help text should be located in the XX/help/ folder.   Refer to the topic on folder locations for additional details.

Name

Text that applies to the entire page, should be named using the same name as the BBx program,  but all lower case.   For example, if the main program is named WA010, then the help text for the main program would be named XX/help/wa010.htm.

Linking Method

Help text that pertains to the entire page can be displayed by the operator by clicking on the "Help" link on the row with other links, typically near the bottom of the page.   This Help link, and the necessary javascript, can be added to your HTML by calling the CDW001 program.

Should you need help text that is field specific, and does not apply to the entire page, you can add the HTML and javascript manually.  

<a href="javascript:cdw001('/xx/help/HelpDocumentTitle.htm')">Field Name</a>

You must also include the CD/web/cdw001 javascript in the HTML.   This can be done by using a bbweb utility.

CALL "uthtmout.wbb","","{include CD/web/cdw001}"+HTML$,HTML$,""

Display Method

The display of the help text is done in a smaller, separate window, that has many controls omitted.   The tool bar control is left visible so the help text can be printed using the printer icon.
