Check Update

 Last Revised: 02/04/14

In order to update the check information to the General Ledger Posting Report, a Check Register must first be printed.   Review the Register for accuracy before updating.

 

It is during the Check Update that postings take place to the Dynamo General Ledger Posting Report. Proceed to print the G/L Posting Report found on the G/L menu.  The G/L Posting Report is an audit report detailing all transactions posted using Invoice Data Entry and Check Printing.  

Check information flows into the Bank Reconciliation program during the Check Update phase as well.
