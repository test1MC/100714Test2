A/P Historical Check Register

 Last Revised: 01/03/14

The A/P Historical Check Register can be used to list all payments for an operator indicated date range.

It provides a recap of all payments processed within the operator-specified time frame with the option to include invoice detail.   Payments processed through both Check Printing and Manual Payment processing are included.  

Itemized information for each payment includes the:

  • Bank Code

  • Check Number

  • Payment Method (blank = printed check, S = separate checks, T = Electronic funds transfer, W = Automated withdrawals,

  • R = Reversals, V = Voided checks),

  • Payment Date

  • Batch type (CD = Check Printing, MP = Manual Payment)

  • Batch number

  • Vendor Number and Name

  • Gross invoice amount, discount amount, and payment (net) amount.

Individual Batch Totals (for both Check Printing and Manual Payment batches) are summarized at the end of the report and includes the number of payments in each batch, batch amounts, and when the batch was updated.
