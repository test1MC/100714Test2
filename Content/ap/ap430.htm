accounts payable Utility Functions

Export Accounts Payable Data

 Last Revised: 01/03/14

Dynamo Accounts Payable provides an export tool to export data into a Tab Delimited file that can be used in a spreadsheet program — Microsoft Excel, for example.

The Export Accounts Payable Data Screen

1 Invoices
2 Invoice Detail
3 Payments
4 Payment Detail

Invoices, Invoice Detail, Payments and Payment detail can be exported directly from Dynamo. 

When the export is complete, Export Accounts Payable Data screen reports the name and location of the exported file so that it can be opened in a spreadsheet.  Normally, the file is named export/your company codeap12.tab. 

Note: .tab is the extension for a Tab Delimited file, which is the type of file this utility creates.
Export Accounts Payable Data
1 Invoices This option exports every invoice in the system, whether paid or not. The exported file includes a sequential voucher number, a PO number, a receipt sequence number (indicating this is the first, second, etc. receipt against this PO), a Vendor Number, the alpha sort for the Vendor, the Terms Code, Invoice Number, Invoice Date, Invoice Amount, Due Date, Discount Amount, Discount Date, the Note or Reference, and On-Hold Indicator, a Payment Method, 1099 information, the Batch type and Number, Date Paid, and Location Number if you are in a multi-location environment.

Refer to Accounts Payable Invoice Data Entry on the A/P Menu for definitions of these fields.
2 Invoice Detail The exported file includes the sequential voucher number; a sequence number; a nine digit, left justified General Ledger Account number; and the amount. 
Note: an invoice distributed to multiple General Ledger Accounts will have multiple listings in this file.
3 Payments All payments on file will be exported to a file named xxap22.tab in the export directory. 

This file includes a sequential payment number, the Vendor Number, and the Vendor's Alpha Sort code. The Sequence Number (what sequence the payment is on the Payment Registry), the Payment Method, the Check Date or Payment Date, the Check Number (if paid by check), the gross amount of the check, Cash Discount taken, the Net Amount of the check, Status Code, Bank Code, Batch Type, the four digit sequential Batch Number, and an Indicator of whether the check has cleared or not.

4 Payment Detail

 For each payment, this option lists the Sequence Number with a reference to the vouchers that were paid and the Gross, Discount, and Net amounts.  For every payment, there will always be at least one payment detail record. With the Voucher Number, you can reference the Invoice file for the Invoice Number and Date, etc.
