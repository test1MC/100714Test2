Dynastart Setup - Windows PC

 Last Revised 11/01/14

Excellware recommends using Dynastart to launch Dynamo.

Dynastart is a version of Dynamo that does not require the use of a terminal emulator—and the associated licensing fees—to run. Instead, Dynastart uses Java, which is a free download and already present on millions of devices. Follow these steps to setup Dynastart on a Windows PC. For best results, the PC should be running Microsoft Windows Professional Edition version 7 or higher. Home Editions and older versions of Windows such as XP or Vista are not recommended.

  1. Uninstall all versions of Java that may have been previously installed
    more Open Windows Control Panel
    Select Programs and Features
    For each entry in the list that does not begin with Java 8, click on the Program, then click uninstall.
    Close Programs and Features, and Control Panel
    Open Windows Explorer (not Internet Explorer)
    Delete folders C:\Program Files\Java and C:\Program Files (x86)\Java if present
  2. Download and install Java
    more Click here to open a browser to the Java download page
    Click the JRE Download Button (Software Developers should click the JDK Download Button)
    Click to Accept the Java License Agreement
    Click the link next to Windows x64 to download the 64-bit version. Older PC's that do not have a 64-bit processors can use the Windows x86 Offline option. You can view Computer System Properties to determine if your System Type is a 64-bit Operating System.
  3. Set the Java Security Level & Add Exception
    more Open Windows Control Panel
    Select Java
    Click on the Security tab
    Select the High security option
    Add the URL used to launch Dynamo in the Exception Site List
