DYNAMO

UPS WorldShip

 Last Revised: 04/01/10

Dynamo can interface with UPS WorldShip.  UPS WorldShip is a program provided by UPS that runs on a PC and is used to compute shipping charges, assign UPS tracking numbers, and print shipping labels.  UPS can obtain theship to address, service (Ground, Next Day Air, Three Day Select, etc.), billing method (Shipper, Receiver, Third Party), and other information from Dynamo.  UPS can also send to Dynamo package weights, tracking numbers, and shipping costs.

This document describes what the shipping clerk needs to do to enable UPS WorldShip to communicate with Dynamo.  It is assumed that UPS WorldShip and Dynamo have already been configured. If not, see the UPS WorldShip ODBC Conversion documentation for help.

If you are replacing a PC that runs UPS WorldShip, you can follow the instructions provided by UPS to transfer the configuration and integration information from one PC to the other without having to manually repeat the configuration process.

Start UPS WorldShip and click on Import/Export Data.  Then click on Keyed Import and choose Dynamo from the list.  If Dynamo is not in the list, click More, and then Dynamo.  If Dynamo is still not in the list, contact Excellware.  Repeat this process each time UPS WorldShip is started.

After clicking on Dynamo, the following window will appear.  You can move this window by dragging the title bar of the window to the desired position.

To start a shipment, if the window shown above does not have the yellow entry box, then click on the window to make it the active window. (If it is not visible because it might be behind another window, hold Alt and press Tab until it is selected). Next scan the bar code on the Dynamo Packing List.  Dynamo Sales Order Packing Lists, Transfer Order Packing Lists, and Vendor Return Packing Lists all have a bar code that can be used.  If you do not have a bar code scanner, you can manually enter the number that prints immediately below the bar code, then touch the Tab key.  Contact Excellware if you would like to purchase a bar code reader that can be used for this purpose.  They cost approximately $167.

If UPS cannot obtain the shipping address from Dynamo, a message box such as the one below will appear.  Contact Excellware for assistance.

Once the bar code is scanned or the number entered manually, the ship to name and address, and service, and shipment billing information should be displayed.  If this information does not display, then contact Excellware for assistance.

If using a scale, then place the package on the scale, and the weight will display, otherwise manually enter the weight.  Contact Excellware to purchase a scale that is compatible with UPS WorldShip.

UPSWorldShip also needs to be configured to send shipment information back to Dynamo.  Click Import/Export Data.  Then click Automatic Export after processing Shipment.  Then click "To Dynamo" from the list.  If "To Dynamo" is not in the list, then click "more".  If "To Dynamo" is not listed, contact Excellware for assistance.

The Automatic Export remains in effect, so you do not have to repeat this process each time UPS WorldShipis started.

Approximately every 10 minutes during business hours, Dynamo checks to see if any shipment information is available from UPS WorldShip.  The package weight, tracking number, and charges can be displayed in the various Dynamo applications including Sales Order Inquiry, Sales Order Processing, Transfer Order Inquiry, Transfer Order Processing, and Vendor Return Processing.  Contact Excellware if the package shipment information is not available for display in Dynamo 10 minutes after the shipment has been processed on the PC.
