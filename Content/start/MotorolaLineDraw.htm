Installing Font with Line Drawing Characters

 Last Revised: 12/18/10

Dynamo Tools use Line Drawing Characters for outlining windows and boxes.  In many cases, the fonts that come pre-installed exclude these Line Drawing Characters.  However, you can easily download and install a Courier New True Type Font that includes these characters, and install on the mobile computer.

If the Line Drawing Characters on the Dynascan menu do not display properly, then it is likely that the installed font lack the Line Drawing Characters.

Click here to download the font file, and here to download a batch file.  Transfer these two files to the mobile device using the Active Sync program and place in the \Application folder.  The presence of the batch (.cpy) file should cause the cour.ttf file to be copied to the \Windows folder.  Select the Courier New font in the emulation parameters, and test the Dynascan menu to verify that the Line Drawing Characters display properly.
