Selecting records from a list

 Last Revised: 05/20/10

Dynamo often presents a list of valid options for an entry.  The list can be automatically presented by the application, or displayed in response to the operator touching the F2 key, or displayed after the operator makes an entry that is not in the list of valid entries.  An example list is shown below.

Note that Dynamo also frequently presents a list of valid options in a grid format.  This is somewhat different than a simple list.  Click here for additional information about selecting records from a grid.

In this example, you may enter an option# (1-10), or move the highlight bar to the desired option and then touch the Enter key, or touch the F4 key to exit.

When presented with a list, the operator has the following options.

Action Normal Key Ctrl + Key Note
Enter the value desired     This would be the option# (1-10) in above example
Move highlight bar down to the next selection. ArrowDn Ctrl + J  
Move highlight bar one column to the right ArrowRt Ctrl +L Only applies when the list has multiple columns
Move highlight bar one column to the left ArrowLf Ctrl + V Only applies when the list has multiple columns
Move the highlight bar to the last selection on the page PageDn Ctrl + N If already on the last selection on the page, PageDn will move the highlight bar to the last selection on the next page
Move the highlight bar to the first selection on the page PageUp Ctrl + O If already on the first selection on the page, PageUp will move the highlight bar to the first selection on the previous page
Move the highlight bar to the last selection on the last page End Ctrl + E  
Move the highlight bar to the first selection on the first page Home Ctrl + P  
Move the highlight bar forward 5 selections Tab Ctrl + I  
Move the highlight bar back 5 selections Shift + Tab Ctrl + B Most operators will need to use Ctrl + B, as Shift + Tab only works in limited cases.
