Selecting records from a grid

 Last Revised: 05/20/10

Dynamo often presents records in a grid format.  The list can be automatically presented by the application, or displayed in response to the operator touching the F2 key.  You can easily identify a grid format by the vertical grid lines that separate the columns.  An example list in grid format is shown below.

Note that Dynamo also frequently presents a list of valid options in a simple list format.  This is somewhat different than a grid.  Click here for additional information about selecting records from a simple list.

In this example, you may enter an option# (1-10), or move the highlight bar to the desired option and then touch the Enter key, or touch the F4 key to exit.

When presented with a list, the operator has the following options.  The workstation will sound the bell if an invalid entry is made.

For workstations with a full keyboard, the 'Normal Key' can be used.  Workstations with a keyboard that does not include the Normal Key specified, can perform the Action by holding the Ctrl key down while touching the Key specified in the Ctrl + Key column.

Action Normal Key Ctrl + Key Note
Move highlight bar down to the next record ArrowDn Ctrl + J If the last record on the page is highlighted, ArrowDn will display the next page with the first record on the page highlighted.
Move the highlight bar down 5 records Tab Ctrl + I If the last record on the page is highlighted, Tab will display the next page with the first record on the page highlighted.

If there are less than 5 records remaining on the page, Tab will highlight the last record on the page.
Move the highlight bar to the last record on the page PageDn Ctrl + N If the last record on the page is already highlighted, PageDn will move the highlight bar to the last record on the next page
Move the highlight bar to the last record in the list End Ctrl + E  
Move highlight bar up to the next record ArrowUp Ctrl + K If the first record on the page is highlighted, ArrowUp will display the previous page with the last record on the page highlighted. 
Move the highlight bar up 5 records Shift + Tab Ctrl + B Most operators will need to use Ctrl + B, as Shift + Tab only works in limited cases.

If the first record on the page is highlighted, Shift + Tab and Ctrl + B will display the previous page with the last record on the page highlighted.

If you are within 5 records of the top of the page, then the first record on the page will be highlighted.
Move the highlight bar to the first record on the page PageUp Ctrl + O If the first record on the page is highlighted, PageUp will move the highlight bar to the first selection on the previous page
Move the highlight bar to the first record in the list Home Ctrl + P  
Display columns to the right of the current view ArrowRt Ctrl +L Only applies when the view does not fit on one page.  The prompt will indicate "ArrowRt=Shift right"
Display columns to the left of the current view ArrowLf Ctrl + V Only applies when the view does not fit on one page.  The prompt will indicate "ArrowLf=Shift left"
Advance the highlight bar to a specific record Text Text As each character is entered, the list will re-display with the first record on the page matching the characters entered.  For example, in the example above where purchase orders are shown in a grid, you can touch the letter C to advance the highlight bar to the first closed Purchase Order, i.e., the one with C in the Opn column.  In most cases, the text you enter is matching the data displayed starting with the first column.

In some cases when the first column is a numeric field, the operator will be prompted: "Enter numeric value to jump to".  In this case the number is entered, then the display is refreshed starting with the first record that has the numeric value entered.
  Enter   If prompt includes: "Enter=select record", select the highlighted record and exit the display.

If prompt includes: "Enter=Continue", you can still navigate the records and columns of the grid, but the highlighted record is not returned for the application to use.
  F1   Temporarily hotkey to another application
  F2   Perform the activity specified in the prompt, i.e., F2=Delete Record.  This option is used infrequently.
  F3   Display this Help Document.
  F4   Exit the display without a record being selected
  F5   Perform the activity specified in the prompt, i.e., F5=Sort by Item#.  This option is used infrequently.
