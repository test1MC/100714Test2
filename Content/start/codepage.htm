Character Set

 Last Revised 06/09/11

The character set, also called a code page, determines what symbols can be displayed on your monitor.  The symbols that vary by code page are typically the special characters including the line-drawing characters that are used to create window borders and boxes, foreign language characters, and many special symbols.  Dynamo displays properly when the character set for your workstation is set to the character set named Cp850.

If the character set is not Cp850, then Dynamo will either make this change for you automatically, or display this page for detailed instructions how you can set the character set.

Windows

The following notes apply to Windows XP, although the steps should be similar for other versions of Microsoft Windows.

  1. Click Start, then Control Panel
  2. Click 'System' if shown in the list of options, otherwise click 'Performance and Maintenance' then 'System'.
  3. Click the 'Advanced' tab on the System Properties application.
  4. Click the 'Environment Variables' tab.
  5. Click the 'New' button in the upper section used for operator variables.
  6. In the Variable name box, enter: _java_options (Note that the first character as well as the character before 'options' is the underscore which is typically located on the keyboard as the shifted hyphen key.  You can copy and paste these values from this document if desired.)
  7. In the Variable value box, enter: -Dfile.encoding=Cp850 (Note that here the first character is really a hyphen, and not an underscore)
  8. Click OK until you return to the Control Panel, then reboot your PC.
  9. If you still get a message upon login to Dynamo to set the Windows character set, then double check the fields above to verify that they were entered properly, or call Excellware for assistance.
Apple Max OS X

Dynamo typically can set the character set automatically, however, you will need to logout of Mac OS X by clicking on the apple icon in the upper left of the display, and choose the Log Out option.  You can log back in, start Dynamo, and should not receive any messages related to the character set.

All Worktations

Note that on all platforms the current character set code is displayed in the Information option of the Dynamo menu which can be accessed by entering "I" at any Dynamo menu.

If the character set is not Cp850 then certain displays including selecting one or more branch or warehouse codes will display the capital letter A with a small circle above instead of the horizontal line character.

Contact Excellware for assistance setting the character set for any other system.
