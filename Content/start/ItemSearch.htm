Searching for Items

 Last Revised: 06/11/11

Dynamo includes a Google like ability to search for items.   Just enter what you know, and Dynamo will provide a list of matching items.

Here are some examples of what you can enter.

You Enter Note
Complete Item# You can always enter the Dynamo Item#, if you know it.  Item numbers ABC123, ABC1234, ABC12345 all begin with ABC123, so when you enter ABC123 Dynamo does not know which of the items that begin with ABC123 you are looking for, so each of these items will be included in the list.  To limit your search to only item# ABC123, touch the Space Bar after entering the item# before touching Enter.  In this case only Item# ABC123 will be included in the list.
Starting Characters of Item# All items that start with the characters entered will be displayed
One or more words from the Item Description Only items that match all the words entered will be displayed.  For example entering RED PEN will display all items that have both RED and PEN in either of the two lines of Item Description.  You may enter the search words in any case (UPPER, lower, Mixed).
Enter or scan UPC Bar Code# Scanning or entering a UPC Bar Code will return the scanned item.  Dynamo requires each UPC Bar Code to refer to a single Dynamo Item#.
Enter or scan Dynamo Bar Code# Scanning or entering a Dynamo Bar Code will return the item associated with the bar code#.  If there are other item numbers or vendor part numbers that start with the same digits as the Dynamo Bar Code, they will be included in the list of matching items.
Starting Characters of Vendor Part# All items that have a Vendor Part# that starts with the characters entered will be displayed. 
Equivalent Item# Any Item that is Equivalent to the entry will display
Customer Part# Only applies when Dynamo knows the customer for the transaction, such as during Sales Order Processing
F3=Help Select 'Item Level Help' to display this Help Text
F5=Search Options You can temporarily change the search options to include discontinued items, or to show only items with a Quantity Available at your warehouse/store, or to show only items that are available at any warehouse/store in the company.
OptionDefault Value
Exclude Discontinued Items with no QOH (Quantity on Hand)Y
Exclude Items if Warehouse Qty Avail=0 (at my Warehouse)N
Exclude Items if Company Qty Avail=0 (Company Wide)N
F4=End Exit Item Search

Since searching for Items can sometimes take a while, especially when many items match the entry, you can also touch the F4 key during the search process to abort the search process, and display the items found so far.
