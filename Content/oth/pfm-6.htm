FacetWin Administration Menu

 Last Revised: 07/16/10

FacetWin is an application that is used by a personal computer to connect to a Linux/Unix server to run Dynamo.  It also provides a way for PCs to access files and folders stored on the Linux/Unix server.  It also provides a way to launch Windows applications such as a web browser from Dynamo.

This application requires root's password.

This application is used to add FacetWin operator licenses, to stop and start FacetWin services, and to view PC Name resolution details.  It is not typically used by end operators.
