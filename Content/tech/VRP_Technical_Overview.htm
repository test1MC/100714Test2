Dynamo vendor returns processing

Technical Overview

 Last Revised: 01/11/08

Dynamo Vendor Returns Processing provides the ability to enter, maintain, and process items that are being sent back to the vendor.

Vendor Returns Data Files

Name Description Usage
XXVR40 VR Heading Record for each vendor return
XXVR41 VR Detail Record for each vendor return line item
XXVR42 VR Timestamps Record for each vendor return timestamp
XXVR43 VR Notepad Text This file contains all notepad text associated with the return.
First key segmentNotepad contents
VR number plus "INTERNAL"Internal notepad not printed on vendor return packing slip
VR number plus "EXTERNAL"Notepad printed at start of VR before first line item
VR number plus HTA(VR41.VR41ID)Notepad associated with line item, printed on vendor return below line item.

Name Description Data Dictionary Company Note
