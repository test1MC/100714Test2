IC033

Return Qty on Hand, Inventory Value & Average Inventory Cost

 Last Revised: 01/05/12

This routine return the quantity on hand, inventory value, and average cost for a specific item on any date.  It can return this information for a specified warehouse, all warehouses, or warehouses of specified types.

IC033 uses the Inventory Receipt file IC03 and Inventory Usage file IC04 to determine the quantity on hand, value, and average cost for an item.

CALL "IC033", Y$, REC$, IC033$

IC033 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
REC$ To

Must be templated field and contain field ITEMNO.
 

If REC$ contains optional field WHS or SHIPWHS, then IC033 will limit results to specified warehouse code, otherwise company-wide results are returned.
 

If REC$ contains optional field WHSTYPES, then IC033 will limit results to warehouses of the specified types.  This is useful when necessary to exclude rental warehouses from total company quantity on hand.
IC033.QOH From Total Quantity on Hand
IC033.VALUE From Inventory Value based on the actual cost of each receipt
IC033.COREVALUE From Core value for systems that track cores
IC033.TOTAL From Sum of Inventory and Core values
IC033.AVGCOST From Computed average cost
IC033.AVGCU From Costing Unit for computed average cost
IC033.HIGHDATE From The latest date where the Qty on Hand changed
IC033.WHSQOH$ From String of warehouse codes and quantities on hand at each warehouse.  The slot length is the length of the warehouse code (typically 1), plus 9 characters for the quantity on hand.  Only returned when a warehouse code was not specified in REC$.
STBL("IC033_DATE") To Optional julian date used to determine the date that the information is based on.  If not set, then the information will be based on today's system date.
