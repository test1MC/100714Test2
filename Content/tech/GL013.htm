GL013

Enter G/L Account#

 Last Revised: 12/07/08

GL013 is an X type external routine that is used to enter or display a General Ledger account number.  The operator can enter an account#, enter a description, or touch F2 to select from a list.

Dynamo General Ledger account numbers are left justified in a zero filled field of 9 characters.

CALL "GL013", ERR=XXXX, MIN, MAX, COL, ROW, PROMPT$, GLACCTLEN, GLACCTNO$, { CTLVAL, OPT$, FMT$[ALL], GLCODE$ }

GL013 Arguments
Field Passed To/From Description
ERR=XXXX N/A Error branch taken if:
a) G/L Parameter Maintenance not yet done, or
b) Unable to open G/L Accounts File XXGL10
c) OPT$ contains DIS or FMT and no G/L accounts have been entered
MIN To 0 indicates entry is optional
non-zero indicates entry is required
MAX To not used
COL To entry column
ROW To entry row
PROMPT$ To null: prompt will be 'Enter G/L Account#, Description, F2=Display'
not null: prompt will be PROMPT$ plus Description, F2=Display
GLACCTLEN From returned length of GLACCTNO$, either zero or 9
GLACCTNO$ To/From default and entered value of G/L Account#
CTLVAL From CTL value of entry, can be 0 (Enter), 2 (ArrowUp), 3 (ArrowDn), 4 (F4)
OPT$ To locnN - no entry of location segment of G/L Account#
locnI - location segment must contain a valid location
locnV - location may vary by location, operator may enter L's in location position to allow the G/L account# to vary by location
NEW, CG , DRV - entry
DIS - display account# and description
FMT - format account# and description, see FMT$[ALL] below
cr=CCRR - display G/L account description @(CC,RR).  Note this expression may precede any of the other OPT$ options shown.
FMT$[ALL] From FMT$[0] - masked G/L account#, i.e., XXXX-XX
FMT$[1] - contains GL08$ location record - (1,4)=Location, (5,30)=Description, (35,10)=Short Description
FMT$[2] - "0" - no entry, "1" - G/L account# entered
GLCODE$ To Pass GLCODE$ to limit entries to the list of valid G/L Account Numbers used during Cash Receipt Application (CRA), Purchase Order Processing (POP), Inventory Transaction Entry (ITE), etc.  The list of valid accounts is maintained using G/L Interface Maintenance (GLU).
