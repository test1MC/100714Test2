PO123

POP Select/Enter Vendor Contact

 Last Revised: 06/29/07

T type external routine used during Purchase Order Processing to select a vendor contact and populate contact related fields in the purchase order header record PO40$.

CALL "PO123", S039$, PO40$, X100$

PO123 Arguments
Field Passed To/From Description
S039$ To All parameters for entry & display
PO40$ To/From Used to populate CONTACT$, EMAIL$, CONPHONE$, CONTYPE$, CONEXT$, CONFAX$.
X100$ To/From T type external routine parameters
