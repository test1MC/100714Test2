IM028

Update Item Activity by Whs

 Last Revised: 07/08/08

This routine is called during SOP Invoice Update and TOP Receipts Update to record shipment data in file XXIM28.

Additional description.

CALL "IM028", Y$, ITEMNO$, WHS$, QTY, { SALES, COST, TYPE$, JDATE }

IM028 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
ITEMNO$ To Item#
WHS$ To Warehouse that shipped item
QTY To Quantity Shipped
SALES To Used for SOP transactions only, indicates the sales amount for item shipped
COST To Used for SOP transactions only, indicates the cost of sales amount for item shipped
TYPE$ To
SOP or null Transaction will update SOP fields in XXIM28 file
TOP Transaction will update TOP fields in XXIM28 file
JDATE To
0 or not passed Transaction will use current sales month and year from XXCW00 parameter file to determine year and month that the transaction should be posted to.  This may differ from the transaction date because the sales analysis month may end prior to the last calendar date of the month as is the practice in many industries including plumbing supply, auto parts, etc.
valid julian date Transaction will use the year and month based on the JDATE passed.
