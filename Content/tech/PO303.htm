PO303

PO Receiving - return list of PO's on receipt

 Last Revised: 06/12/08

This CDS037/38 style called routine is used to retrieve a list of purchase order numbers included on a PO Receipt.

CALL "PO303", Y$, PO50$, PO303$

PO303 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
PO50$ To PO Receipt Heading
PO303.FIRSTPO$ From 8 character field containing first PO
PO303.POS$ From 18 character field containing first PO and the count of additional PO's on the receipt
PO303.POLIST$ From variable length field containing list of 8 character PO numbers included on the PO Receipt
