IC010

Return Price/Cost Unit Quantity

 Last Revised: 03/16/09

This routine is called to determine the quantity associated with any costing or pricing unit.  The costing/pricing unit is a single character field such as blank for each, C for per hundred, M for per thousand.  This routine can also be called to obtain a list of the valid costing units.

Additional description.

CALL "IC010", CU$, CUQTY

IC010 Arguments
Field Passed To/From Description
CU$ To/From If passed as a single character costing or pricing unit, then CUQTY will be returned as the numeric value of the unit.  If passed as a null string, then CU$ will be returned as a pipe separated list of valid costing units and related descriptions.
