PO301

POP Select PO Receipt

 Last Revised: 05/29/08

Routine called to return PO Receipt heading record based on operator entering a receipt#, selecting from a list, or a new receipt heading record.

CALL "PO301", Y$, PO301$, PO50$

PO301 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
PO301.TYPE$ To
blank Display All Receipts
O Display Open Receipts only
N Enable option to create new PO receipts
PO301.COL To Column number on display where receipt# is entered/displayed.
PO301.ROW To Row number on display where receipt# is entered/displayed.
PO301.VENDNO To If passed as non-blank, then PO301 will restrict entry and selection to receipts linked to this vendor.
PO50$ From Selected PO Receipt heading record.  PO50.RCPTNO will be zero if no record is selected.
