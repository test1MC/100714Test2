PO530

POP Write Receipt Timestamp

 Last Revised: 05/29/08

This program is called during various Purchase Order Processing activities to create a new timestamp record in file XXPO53.

The Timestamp record will use the login ID of the operator currently logged in, and the current date and time.

CALL "PO530", Y$, RCPTNO$, NOTE$

PO530 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
RCPTNO$ To Receipt# from PO50.
NOTE$ To Note describing activity such as "Started new receipt", "Started Scanning", "Updated Receipt", etc.  49 characters max
