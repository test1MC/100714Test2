PO420

POP Write Timestamp

 Last Revised: 05/29/08

This program is called during various Purchase Order Processing activities to create a new timestamp record in file XXPO42.

The Timestamp record will use the login ID of the operator currently logged in, and the current date and time.

CALL "PO420", Y$, PONO$, NOTE$

PO420 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
PONO$ To Purchase Order#
NOTE$ To Note describing activity such as "Created New PO", "Closed PO", "Faxed PO", etc.
