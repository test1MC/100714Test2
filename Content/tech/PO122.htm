PO122

POP Select Vendor

 Last Revised: 06/29/07

This is a T Type external routine used during Purchase Order Processing for selection of vendor.

Calls PO138 for mapping of vendor fields to purchase order header.  Will set CDS550 global to bypass entry of address and telephone number fields for non-miscellaneous vendors.

CALL "PO122", S039$, PO40$, X100$

PO122 Arguments
Field Passed To/From Description
S039$ To All parameters for entry & display
PO40$ To/From Purchase Order Heading record will be populated with vendor#, name, and other vendor information.
X100$ To/From Standard T Type Routine
