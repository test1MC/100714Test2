LP500

PCL Label Printing

 Last Revised: 03/02/09

This routine can be used when printing labels on sheets for laser printers.

LP500 supports 8 1/2 x 11" sheets containing one or more columns and rows of labels.  A typical address label sheet has 3 columns with 10 rows with a 1/2" top and bottom margin on the page.

After printing the first page of labels, the operator will be prompted to print the next page, print all pages, or abort the application.

This application requires the LP500 data dictionary from company DY.

CALL "LP500", Y$, Y6$, LP500$

LP500 Arguments
Field Passed To/From Description
Y$ To Session Control Variable
Y6$ To Printer Control Variable
LP500.HEADING$ To Optional text that will print in the top margin of each page, such as "Customer Labels", "Vendor Labels", etc.
LP500.LINE$[10] To Up to 10 lines of text that will print on the label.  Blank lines are ignored.  The font size of long text will be reduced in order to fit the entire line on the label.  Labels consisting of no non-blank lines will not be printed.  The vertical spacing between the rows is determined by the number of lines printed for each label.

LP500.ALLCAPS$ To Pass as "Y" to have LP500 capitalize all letters on all lines.
LP500.ROWS To Pass as the number of rows of labels per page to override the default of 10 rows per page.
LP500.COLS To Pass as the number of columns of labels per page to override the default with 3 columns per page.
LP500.FONTSIZE To Pass as the font size to override the default font size of 10 points.
LP500.REPEAT To Pass as the number of labels to print per call to LP500 to override the default of 1 label per call.
LP500.COUNT N/A Used by LP500 to track the number of labels already populated on the page.
LP500.MODE N/A Used by LP500 to manage operator prompts at the beginsning of the job.
LP500.TOPMARGIN N/A Used by LP500 to set the top margin.  Inkjet printers have a slightly different top margin than laser printers.
Example with 30 and 40 characters per line and 1 to 8 lines per label
1000 BEGIN
1005 CALL "CDS084",Y$,Y5$,U0,S084$,Y6,Y6$; REM "Open Printer
1010 CALL "CDS041","LP500",LP500$,"YY"; REM "Retrieve template
1015 LET LP500.HEADING$="LP500 example"
1020 FOR TEXT=30 TO 40 STEP 10
1030 FOR LINES=1 TO 8
1040 FOR LINE=1 TO LINES
1050 LET LP500.LINE$[LINE]=PAD(STR(TEXT)+" characters/line, "+STR(LINES)+" lines ",TEXT,"*")
1060 NEXT LINE
1070 CALL "LP500",Y$,Y6$,LP500$
1080 NEXT LINES
1090 NEXT TEXT
1100 CALL "CDS094",Y6$,"E"; REM "Close printer
