Transfer Order Processing

 Last Revised: 01/08/14

With this application, the operator can create new and manage existing transfer orders.

Transfer Order Processing Options
(N) Start New Transfer Order

Dynamo assigns a new transfer order number. Fill in the necessary information.

(O) Enter Existing Order# Choose from the display list or type an existing order number.
(B) Scan Bar Code Scan the bar code or enter the number onto the screen.
(T) Display Transfers for Specified Shipping Warehouse Choose an order based on the shipping warehouse.
(F) Display Transfers for Specified Receiving Warehouse Choose an order based on the receiving warehouse.
(A) Display All Orders Choose an order from the complete list of orders.

 

Transfer Order Processing Menu

Once a transfer order has been selected, a list of options displays. Each of these options is marked with an asterisk '*', the letter 'A', 'D', 'S', or a blank:

  • An asterisk '*' indicates there is data available to display for this option.
  • The letter 'A' indicates that no data is available, but you may select this option and enter information.
  • The letter 'D' indicates this option is disabled.
  • The letter 'S' indicates you do not have sufficient security clearance for this option.
  • A blank next to an option indicates there is no data to display.
Options
1. Heading Edit heading information, including the shipping/receiving warehouses, the priority, the order date, the weight, the ship date, the carrier, the service, the billing method, the due date, the received date, and the number of cartons.
2. Detail Update/view the items to be shipped for this transfer order.
3. Print/Fax/Email

Choose to print, fax, or email the transfer order packing list or the transfer order receiving document.

 

Sample documents:

  • Packing List
  • Receiving Document
4. Quantities Shipped & B/O View the amount of each item ordered, shipped, and backordered in this transfer order.
5. Ship Using Workstation Scanner Use Dynascan (if available) to ship the transfer order.
6. Timestamps

View when any changes were made to the transfer order, what was changed, and who made any changes.

7. Shipment Tracking Track the shipment of the item. A link to the UPS website will open to allow you to view the status of the order.
8. Merge Detail From another Order Merge current transfer order with another by:
  1. Entering an Existing Order#
  2. Scanning the Bar Code
  3. Displaying Transfers for Specified Shipping Warehouse
  4. Displaying Transfers for Specified Receiving Warehouse
  5. Displaying All Orders
9. Update/Cancel/Re-Open/Delete Update, cancel, re-open, or delete the transfer order.
