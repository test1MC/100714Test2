Transfer Order Generation

 Last Revised: 01/08/14

This application is used to create transfer orders based on inventory levels; if some warehouses are overstocked and some are understocked, Transfer Order Generation will recommend transfer orders between the warehouses. The application compares the Stock Status Quantity to a level in between the maximum and minimum amount of inventory for a particular item. The application will fill the first listed warehouse first and continue in order. It will also round to the nearest case size and the nearest package size.

  1. Choose the shipping warehouse.
  2. Choose the receiving warehouse.
  3. Specify the sequence.
  4. Specify the range.
  5. Indicate what should be done with the insufficient stock.
    1. Exclude item from order
    2. Backorder item
  6. Specify the minimum and maximum percentage.
  7. Indicate if the listing should be printed or not.
Format Options
(N) Do not generate TO

Use this option to cancel generation of the transfer order and just view a listing.

(G) Generate TO Use this option to generate your transfer orders without printed copies.
(P) Generate and Print TOs Use this option to generate your transfer orders and receive printed copies of them.
