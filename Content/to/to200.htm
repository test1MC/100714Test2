Transfer Order Listing

 Last Revised: 04/18/14

Transfer Order Listing allows you to print a listing of transfer orders created within Dynamo. It will display the transfer order number, the status (new, shipped, received, or cancelled), the shipping and receiving warehouses, the order date, the shipped date, the received date, the reference (if applicable), the items ordered, the ordered value, and the shipped value.

  1. Indicate what you want to print:
    1. All transfer orders
    2. New (not yet shipped) transfer orders
    3. Shipped (in transit) transfer orders
    4. Received transfer orders
    5. Cancelled transfer orders
  2. Specify whether or not you want to include all order dates.
  3. Select all desired warehouses for shipping and for receiving.
  4. Indicate if you want to print a summary or detail listing.
