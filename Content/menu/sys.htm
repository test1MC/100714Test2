Main Menu

Dynamo Menus

 Last Revised: 12/28/13

This is the initial Dynamo menu.  All Dynamo applications can be reached from here. To reach this menu from any other menu in Dynamo, type SYS.

It is not required that all operators have access to the Main Menu. Operator Preference Maintenance (SMS) can be used so that the initial menu displayed is a preferred menu.  If an operator does not have an Operator Preference setting for 'Menu Code at Login,' then this Main Dynamo Menu will be displayed upon login.
