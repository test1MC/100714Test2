Inventory Control Reports

ICR Menu Overview

 Last Revised: 02/16/15

The Inventory Control Reports (ICR) menu provides the operator with different reports necessary for evaluating inventory. Warehouse Location Labels can also be printed from this menu.

  1. Inventory Evaluation Report
  2. Inventory Usage before Receipt Listing
  3. Negative on Hand Listing
  4. Excess Inventory Report
  5. Quantity on Hand Audit Report
  6. Warehouse Location Listing
  7. Warehouse Location Labels
