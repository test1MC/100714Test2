SMS Menu Overview

 Last Revised: 12/28/13

The applications on this menu are suitable for operators of all security levels.

The SMS applications can be used to:

  • Display active session details
  • Print, Delete, and manage reports printed using the Dynamo Print Spooler
  • Monitor and test printers
  • Print, Delete, and manage reports archived in Dynastore
  • View faxing status and managed faxed documents
  • View email log
  • View and manage reports printed to a file included spreadsheet compatible files
  • View and manage Dynamo messages
  • Set personal profile preferences including Dynamo password
  • View User Count statistics
  • View Server Backup log
  • Use a simple calculator
  • Access other menus for fax & printer management and system configuration
  • change their Dynamo login passwords, display server backup logs, use a simple calculator, and manage Dynamo messages.
