Item Maintenance Utility Functions

Menu Overview

 Last Revised: 02/16/15

This menu includes options to manage item records. These applications do not need to be run regularly; they are only needed for item changes or deletion.

  1. Duplicate Item Record
  2. Item Deletion
  3. Change Item# Entry
  4. Change Item# Listing
  5. Change Item# Update
  6. Warehouse Location Label Printing
  7. Rebuild items by keyword file
  8. Clear Quantity on Hand Audit Data
  9. Change Vendor# in Item File
  10. Alternate Item Analyzer
  11. Item Files Cleanup
