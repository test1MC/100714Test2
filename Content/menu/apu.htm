accounts payable Utility Functions

Menu Overview

 Last Revised: 02/18/15

The programs on this menu are used to configure Dynamo Accounts Payable, to specify terms codes, to manage banking details (including transaction reconciliation), and to export Accounts Payable data.

  1. A/P Parameter Maintenance
  2. Terms Code Maintenance
  3. Bank Code Maintenance
  4. Bank Reconciliation
  5. Bank Balance Inquiry
  6. Invoice Batch Control Check
  7. Export Accounts Payable Data
  8. A/P Invoice Update - G/L Only
