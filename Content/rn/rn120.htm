Rental maintenance & listings

Rental Code Listing

 Last Revised: 01/07/14

This application allows the operator to view created rental codes based on the effective date. Rental codes are created in Rental Rate Maintenance.

Rental Code Listing Formats
Description

This is a report listing items by rental code with their descriptions.

Items

This listing contains the rental code, the description, the item number, and the item description.

Rates

This option requires the operator to specify an effective date. The listing contains the rental code, the description, and the daily, weekly, monthly, and usage rates. It will also display the type.
