Customer maintenance Utility Functions

Ship to Address Listing/Purge

 Last Revised: 09/06/13

This application allows the operator to print all available customer shipping addresses or purge outdated ones.

After the operator allows the application to count all addresses for each year on file, three options become available.

Options
1. Print all addresses on file All of the ship-to addresses entered into Dynamo at any point will be printed. The operator can view these in order to set up mailing or to decide which addresses are no longer necessary.
2. Print all old addresses The operator specifies a "not used since" date to indicate which addresses are old; those will print.
3. Purge old addresses The operator specifies a "not used since" date to indicate which addresses will be deleted from Dynamo.
