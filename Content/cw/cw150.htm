Company Wide Parameters

Document Message Maintenance

 Last Revised: 09/17/13

This program allows the operator to maintain these company wide messages that will appear on certain documents within Dynamo:

  1. POS invoice message
  2. SOP invoice message
  3. Returns policy message
  4. Promotion message
  5. Rental agreement message
  6. Repair service message

Choose any of the previous items from the menu to edit the message; simply type in the provided notepad after choosing the item.
