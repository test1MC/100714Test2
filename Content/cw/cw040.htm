Application Menu Maintenance

 Last Revised: 01/10/14

This program allows the operator to customize the application menus throughout Dynamo. Typically, the operator for this application would be a software developer who wants to remove non-applicable options. Alternatively, an administrator may use this application to restrict access by assigning security levels to certain programs.

Application Menu Maintenance
Program Enter the program number to be maintained.
Option Title Enter the option from within the application to be maintained.
Program The name of the program will be listed here.
Print Enable this option if the program will allow printing.
Sec Level Indicate the security level needed to utilize the associated program. Security levels are defined in Menu Control Maintenance.

Choose the program name and edit the information below it by using the arrow keys, the DELETE key, and the INSERT key. The operator must also indicate whether or not that program supports printing by typing in Y or N.
