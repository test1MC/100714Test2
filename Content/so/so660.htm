Sales by Branch

 Last Revised: 04/18/14

This program allows the operator to view and print a listing of sales by branch. It displays the last week of sales, the sales for the current month, the sales for the previous month, the sales for the current year, and the sales for the previous year for each branch.

Type in the date and select which branch(es) will be in the listing.
