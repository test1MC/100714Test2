sales order processing Utility Functions

Assign/Delete Sales Order Numbers

 Last Revised: 07/23/13

If an operator's company has a pre-numbered list of order forms, with this program, the operator can assign starting sales order numbers, assign ending sales order numbers, or delete any sales order numbers in between. If the operator plans to use all of these orders, he or she should assign them. If a sales order has been voided or damaged beyond usability, the operator should delete that number.

When selecting Print Unused Sales Order Numbers, you can view which numbers you have available out of your assigned sales order numbers.

The operator must enter "A" to assign sales order numbers.

The operator must enter "D" to delete sales order numbers.
