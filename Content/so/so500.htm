Sales Order Booking Report

 Last Revised: 07/11/13

The Sales Order Booking Report is used for viewing of the sales that were made—not necessarily shipped or invoiced—by salespeople in a specified booking month. This can be used to track a salesperson's sales for a particular time period.

 

Choose the salesperson sequence (salesperson (SLS), alpha sort name & salesperson, or login ID), the salesperson range, the month & year, whether or not each order number will show (detail or summary), and the branch(es). If you choose to show each order number, the report will show a summary for each salesman and then a summary by branch. This application can be run at any time for a current or previous period.
