Rental Transaction Listing

 Last Revised: 04/18/14

Rental Transaction Listing is used to display the revenue generated for a group of rental items in a given range. The operator may use this report to view the cost of the fuel charge for customers for a particular time period or to view the adjustments made in the previous month, for example.

Operator-Specified Fields
Starting Month & Year Specify the beginning date to be included on the listing.
Ending Month & Year Specify the ending date to be included on the listing.
Branch Choose the desired branch.
Product Class Type "Y" to select a specific product class or "N" to include all of them.
Item 

Choose one of the following:

  • All Items
  • All Items on Item File
  • All Items not on Item File
  • Operator Specified Item

 

If "All Items" is selected, the report may take a few minutes to generate depending on how many items are in the system. If the operator prefers to see transactions for one item, "Operator Specified Item" should be chosen.

Customer Type "Y" to select a specific customer or "N" to include them all. If all customers will be on the report, it may take a few minutes to generate.
Customer Type Type "Y" to select a specific customer type or "N" to include them all.
Sequence

Choose one of the following information selections:

  • Branch+Warehouse+InvoiceDate+SalesOrder#
  • ProductClass+InvoiceDate+SalesOrder#
  • CustType+Customer+InvoiceDate+SalesOrder#

 

The report displays the alpha code, the order number, the product class, the item number and description, the daily rental cost, the weekly rental cost, the monthly rental cost, the overall usage, the fuel costs, the adjustment costs, and the total. The rest of the information displayed is based on what is chosen as the sequence.
