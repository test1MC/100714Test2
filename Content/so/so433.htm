Month End Procedures

Change Order Booking Month

 Last Revised: 03/18/14

The application should be run every month as soon as the last order for the current month has been entered AND before entering the first sales order for the new month. The order booking month controls only affect the Sales Order Booking Report. Booking an order means that a sales order is being recorded in Dynamo.

Note: Invoices for the prior month can be printed and updated even after changing the order booking month to a new month.
