Zip/Address/Phone Lookup

 Last Revised: 07/23/14

You can use this application to:

  1. Look up the 5-digit zip code and county for any US city,

  2. Look up the city for any 5-digit US zip code,

  3. Validate any US or Canadian address, including verification of street number, street name, city, US Zip+4, and Canadian Postal Code, or

  4. Provide the address for any phone number listed in the White Pages anywhere in the US.

     

    Note: If this application can't seem to validate your street address, please follow these guidelines:
    • If there is only one address line, enter it on Address Line 1.
    • Any suite or apartment number needs to be on the same line as the address.
    City Name

    Type in a city name to find its corresponding zip code, or select it from the list.

     
    Zip Code

    Type in a zip code to find its corresponding city name, or select it from the list.

     
    Address

    Type what you can remember from a street address, and Dynamo will correct it for you.

     
    Phone#

    Type in the phone number for a particular company/person, and Dynamo will provide the address for you.
