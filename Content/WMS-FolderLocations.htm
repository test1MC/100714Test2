Web Management Software

The web server has a folder known as "server root".  This is the highest level directory of the web server.  Below the "server root" folder, there is a folder for documents and cgi scripts.   If a browser URL is pointed to the server name, without any directory or file name, then the document named index.html is the Documents Folder is displayed.

Web Server Typical Server Root Folder Documents Folder cgi folder
Lotus Domino (AIX) /usr/lpp/internet/server_root pub cgi-bin
Apache /usr/local/apache htdocs cgi-bin
Netscape (SCO OpenServer 5) /usr/internet/ns_httpd/httpd-80 htdocs cgi-bin

The Web Management Software requires a folder below the Documents Folder with a folder name of the lower case version of the company code.   This is represented as xx in the table below.   These folders typically link to a folder below the base SMS directory (SMSDIR), typically /u/CDI.

Folder Name Contents Links to
xx/help/ help text in html format (SMSDIR)/XX/help/
xx/css/ Cascading Style Sheets (SMSDIR)/XX/css/
xx/images/ images and graphics used on all pages (SMSDIR)/XX/images/
