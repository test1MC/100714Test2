Inquiry Functions

Serialized Item Inquiry

 Last Revised: 01/08/14

This application can be used to display information about a particular serialized item.  It includes both items for sale and items for rent.

Record selection is done by entry of Serial#, Rental ID#, or by entering an Item# and selecting a serial# from the list.

Serialized Item Inquiry
1. Rental History View the sales order number, the date out, the date in, the total cost, the branch, the warehouse, and the customer for each time the current serialized item was rented.
2. Print Serialized Item Label (This option is enabled only when an ELTRON printer is specified.) The printed item label contains a bar code and pertinent serialized item information for rental items.
