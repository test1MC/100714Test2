Serialized Inventory Subsystem

Serialized Item Listing

 Last Revised: 08/30/13

Serialized Item Listing is used to display a listing of serialized inventory. The list will contain the item number, the description, the warehouse, the rental ID number, the serial number, and the status (when applicable).

The operator can view:

  • (A) All Serialized Items
  • (S) Serialized Items On Hand
  • (D) Disposed Serialized Items
  • (R) Rented Serialized Items

The operator must select the item sequence and range as well as the warehouse(s).
