accounts receivable Utility Functions

Audit Customer A/R Control Totals

 Last Revised: 07/29/13

Audit Customer A/R Control Totals is used to verify whether or not invoices add up to the customer control total. If these two are not balanced, this application prints the information. An operator may need to run this program if the power went out, the computer locked up, or any other technical issues occurred during a Cash Receipts Update. If a Cash Receipts Update has been interrupted, Audit Customer A/R Control Totals may display that the total does not agree with the sum of the detailed transaction. If this is the case, the operator can view the print-out and then run Reset Customer A/R Control Totals.
