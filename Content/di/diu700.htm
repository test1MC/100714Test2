customer maintenance utility functions

Change Customer# Entry

 Last Revised: 02/05/14

Change Customer# Entry is used to change a customer's number that has been assigned by Dynamo, as these cannot be changed in Customer Maintenance. After using this application, a listing of all customer number changes can be viewed in Change Customer# Listing. To apply these changes, the operator must use Change Customer# Update.

This application may be used when a customer is combined with another customer (due to a merger, for example). The payment, sales order, invoice, contact information, and other historical data is retained when changing the customer number instead of creating a new number for the customer.
