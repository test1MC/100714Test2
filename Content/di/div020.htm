Cash Receipts Entry

 Last Revised: 10/05/15

In this program, the operator records customer payments.

The operator must enter:

  • the customer name
  • the date the check was received
  • the check number
  • the check amount

If all checks were received on the same day, enter one date upon opening the application. If you are entering checks that were received on multiple dates, use the space bar to enter separate dates for each check. Note that it is important to enter the correct date the check was received, not deposited, so that Dynamo can accurately calculate the customer's average time to pay after an invoice has been sent.

Upon completion of this application, the operator can use the Cash Receipts Listing & Deposit Ticket application. The checks and the deposit ticket can then be taken to the bank. Please view our deposit ticket guidelines that describe which deposit tickets are compatible with Dynamo.
