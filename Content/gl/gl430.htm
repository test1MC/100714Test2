General Ledger Utility Functions

Export General Ledger Data

 Last Revised: 02/05/14

Export General Ledger Data is used to reconcile balances and transactions that were applied by General Ledger account over an operator-specified date range. After running the program, a spreadsheet listing is displayed based on which format the operator chose.

For account balances:

  1. Format - Account Balances
  2. Fiscal year

For transactions:

  1. Format - Transactions
  2. Starting date
  3. Ending date
Formats
Account Balances

This listing contains:

  • the general ledger account number
  • the general ledger description
  • the beginning balance
  • the general ledger information from each month end of the specified fiscal year
  • the ending balance
Transactions

This listing contains:

  • the sequence number
  • the specified dates
  • the source
  • the batch number
  • the general ledger account number
  • the reference (sales order processing, nightly inventory update, etc.) 
  • the amount of each transaction
