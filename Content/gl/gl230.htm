General ledger reports

G/L Transaction Report

 Last Revised: 02/05/14

The General Ledger Transaction Report provides a detailed list of all G/L postings for the operator-specified range of dates and G/L account numbers. It is not necessary to run this report unless the operator desires to know the detail about various transactions that were posted to general ledger accounts.

 

For each G/L account, the account number, account description, and the beginning balance on the starting date specified are displayed. Each G/L transaction including date, source code, batch#, posting reference, debit/credit amount, and running balance is shown. The report total includes totals for the debit and credit columns.
