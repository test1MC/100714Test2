General ledger Utility functions

Delete Updated G/L Posting

 Last Revised: 02/11/06

This feature provides the option to delete a G/L posting that was updated in error.

Although many accountants may disagree with a program such as this which can delete an updated posting, it can be useful when a posting has been made with the incorrect posting date. 

Note: the posting is permanently deleted and no transaction details will appear in G/L Inquiry nor the G/L Transaction Report.
