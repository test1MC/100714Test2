General ledger Utility functions

G/L Location Maintenance

 Last Revised: 01/07/14

The General Ledger Location Maintenance program is only applicable for those companies who have multiple sites. It is used for setting up, adding, changing, or deleting location codes and description.

The G/L Location Maintenance Screen
G/L Location Maintenance View
Locn (Location) Create a number code to represent the physical location.
Description Give a full description of the physical location.
Short Desc Give a brief description of the physical location.
