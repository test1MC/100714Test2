System Management Configuration

Basis License Manager

 Last Revised: 02/11/06

This option executes the Basis Install Main Menu.

This program is used to install any of the Basis program components.  These include: License Manager, Pro5 Business Basic Language, the Pro5 Data Server.

This program can also be used with BBj in systems where BBj has been installed.

This program requires root access, and the operator will be prompted for Root's Password to access the program.
