Back Order Fill Report

 Last Revised: 02/06/14

Back Order Fill Report is used to display any back-ordered items that have been received since they went on back order. The listing can be printed in order to prompt someone to fill an order. The listing displays the date, the order number, the customer, how many back-ordered items there were, as well as how many back-ordered items can be filled at this time.
