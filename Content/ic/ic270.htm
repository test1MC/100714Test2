Quantity on Hand Audit Report

 Last Revised: 02/06/14

Quantity on Hand Audit Report is used to view inventory transactions in a specified range. The report will display the item number and description, when it was posted, the transaction date, the type of order, the order number, the alpha name of the customer, the before quantity, the quantity added or subtracted, and the after quantity. The operator must specify the item sequence, the range, the warehouse(s), the start date, and the end date.
