Negative on Hand Listing

 Last Revised: 02/06/14

This program allows the operator to view and print a negative on hand listing, which is a report of the items that were sold before they were receipted. The list will display the item number and description, the location, the decrement date, the reference number, the type, and the negative on hand amount.

Select which warehouse(s) you'd like to display on the report.
