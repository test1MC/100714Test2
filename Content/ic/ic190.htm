Warehouse Location Labels

 Last Revised: 01/07/14

This program allows the operator to print warehouse location labels based on a selection of items. An operator can also print slightly different warehouse location labels from the IMU menu at Warehouse Location Label Printing.

Indicate:

  1. The item sequence
  2. The range
  3. The location range
  4. The warehouse

After accepting these specifications, this message on the bottom of the screen will display:

Insert the appropriate label sheets to print the labels.
