Server Backup Log

 Last Revised: 11/22/08

If using the Excellware Server Backup routine to copy data from the server to a tape each night, then this program can be used to display the log of each backup attempt.  It is similar to the email sent each evening which contains the last 30 backup attempts.

The operator can request a list of all backups, or only those that have a 'Bad' status.  Refer to the Server Backup document for more details.
