Printer & Fax Management

Display Fax Log

 Last Revised: 02/11/06

This program is provided by the Vis*Fax Software.

Display Fax Log displays similar information to the Display Fax Log & Options screen, but lacks some of the functionality and is not as comprehensive.

Note: because this is not a Dynamo program, the operator must enter Q to quit to get out of the program and return to the Dynamo Menu.
