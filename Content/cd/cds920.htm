System Management Utility Functions

Display Added Programs

 Last Revised: 05/23/14

This program will display or drop those programs that occupy memory.  It may be run from the menu or called.  When called with no arguments, all programs that occupy memory space outside of the program workspace are displayed along with the size of the program in characters and pages (one page equals 256 characters) and an indication if the program has been added permanently using the ADDR command. Totals are displayed.

The Display Added Programs Screen
Technical Notes

When called with a parameter, the valid parameters are:

  • "D" to drop all programs, or
  • "d" to drop all programs that do not begin with CD.
Note:

This program is called with the "d" option by the selector whenever the operator changes to a new company. This is done to insure that a called program will be found in the directory as specified by the prefix and not one that remained in memory from a different directory from another company.
