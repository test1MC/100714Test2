System Management Utility Functions

Dynascan Menu Simulator

 Last Revised: 05/23/14

Dynascan is a collection of Dynamo applications that run in a hand-held computer with an integrated bar code scanner.  Dynascan applications can be used to receive product, put-away product, pick products, pack products, record physical inventory, etc.

This application is used to simulate what the operator would see on the hand-held scanner (though narrower than shown in Dynamo). It is useful for demonstrating or testing Dynascan applications.
