Data Dictionary Printing

 Last Revised: 05/03/07

Prints contents of Data Dictionary for operator specified criteria.

Operator Entries
Detail/Summary/Programs
D Detail - Report includes key structure, field information, and file conversion details.
S Summary - Report includes File Name, Description, File Type, Menu, and File Conversion Summary.
P Programs - Report lists all external programs with a cross reference of what data dictionary entries refer to an external program
Sequence The report can be printed in either File Name or Menu Code sequence.
Range Operator may specify range of File Names or Menu Codes depending on Sequence selected
File Selection Option
1 Includes all files
2 Includes only files where the data dictionary differs from the current data file structure.
3 Includes data dictionary entries for data files, excluding entries for temporary work files and programs.
Include File and Field Notes Option to include any file and field notes on the report.
