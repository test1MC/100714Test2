System Management Utility Functions

Save/Restore program security levels

 Last Revised: 02/11/06

The customer maintains his own security level codes and company name and address.  This program is used to save menu security information when the menu file is being replaced with a copy from Excellware.

New programs are added to the selector and the selector file is copied to the customer's system.  This destroys any program security level codes that the customer may have entered.  In order to prevent this situation from occurring, this program will save these security level codes prior to replacing file XXS150 with a copy from Excellware.  Then after the file has been replaced, it will restore the saved security level codes.

Only level codes above 1 are saved.  Even if the program has been moved from one menu to another, the level code can be properly restored.  Each program is identified by all 6 characters of the program name, plus the screen suffix and the pass parameter.  Therefore, if the new selector file has a different pass parameter, for example, the level code will not be properly restored.  In most cases the selector file has been changed due to new additions and not changes to existing entries on the menu.

In addition to saving the security levels, this program saves the program memory pages, and the hot key selection code.

The saved information is kept in file XXSD00 with a key of SECLV1XXX and removed after the restore has been completed.

The Save/Restore Program Security Levels Screen
In summary
  1. Create tape/disc at Excellware with XXS150 - Selector file.
  2. On Customer's system, run this program to Store all security levels as well as dates, company name, etc.
  3. Restore tape or copy disc file, replacing customer's XXS150 file. Security Level codes have now been replaced, losing any customer entered changes.
  4. Re-run this program to Restore all security level codes, dates, company name, etc.
Technical Notes

This program may be called. The parameters are Y$ - standard terminal control string. Used to determine what company code to process. MODE$ - "S" to save security levels, "R" to restore them.
