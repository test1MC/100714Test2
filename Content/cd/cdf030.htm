VDT Format Maintenance

 Last Revised: 07/16/10

This application is used to design Video Display Terminal (VDT) Formats.  The Dynamo menu processor can display the lower intensity (background) text stored using this application.  The Documentation Manual Printing can be used to print these VDT formats.

Although the use of VDT Formats has been phased out, they are still supported, and a few applications still use them.
