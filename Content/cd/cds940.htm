Data Dictionary Company Update

 Last Revised: 05/23/14

This program is used when it is necessary to update the data dictionary for a specific file on multiple companies on the same server.

Starting in the company that has the updated dictionary, select this option from the menu and enter the file name to be exported.  The transfer will include the data dictionary, views, and file maintenance configuration data.  The operator will be prompted with each company code that has the file in its data dictionary and given the opportunity to revise the data dictionary.

If the dictionary is a data file and the data file differs from that stored in the data dictionary, then the fields to be added, changed, or deleted, as well as title and mask changes will be displayed for each company.  The operator can choose to archive and convert the data file tFo match the revised data dictionary.
