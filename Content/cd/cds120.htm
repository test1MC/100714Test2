Data Dictionary Compare

 Last Revised: 05/03/07

This program can be used to compare a data dictionary entry or template file to another data dictionary entry or template file.  It can be used to compare data for the same file between two different companies, different files within the same company, or different files between different companies.

Only the field characteristics such as field name, template type, title, and mask are compared.  Field verification and key structures for example, are not currently compared using this program.

Operator Entries
Compare From Company Code and File/Dictionary name, i.e., DSCM01, CM040., etc.
Type If a template file exists, the operator will have the option to specify T to use the template file for comparison.  If a data dictionary entry exists, the operator will have the option to specify D to use the data dictionary entry for comparison.  Note that the name of the Data Dictionary Field Information File (DD11) will display when using the D option.
