User Count Statistics

 Last Revised: 05/23/14

Several components of the Dynamo software are based on operator licenses.  This program is used to display operator Count Statistics to determine if the number of licensed operators is sufficient to meet the needs of your organization.

O/S operators are Operating System operators.  Some of the older operating systems such as SCO Unix, and AIX were licensed based on the number of simultaneous operators.  Since 2000, Excellware has been using the Linux operating system which does not have any operator license restrictions or fees that are based on the number of operators.

BBx operators are Business Basic operators.  Dynamo is written in the BBx programming language, and BBx is licensed based on the maximum number of simultaneous operators.  A BBx license is needed for each of the following:

  • each operator logged in,
  • one for Dynamo System Administration which is always being used,
  • one for certain other periodic processing including updating the Fax Log,
  • one for each PC that runs the UPS WorldShip, Federal Express Shipment Manager, or other PC based shipment processing application that uses ODBC to connect to Dynamo Data,
  • one if an operator uses Microsoft Excel or other application to access Dynamo data using ODBC,
  • one for each additional operator session, except when using the FacetWin terminal emulator, which only consumes on BBx license, regardless of the number of Dynamo sessions opened on a single PC,
  • one license is required each time a Dynaweb operator clicks the mouse.  This license is typically checked out and back in very quickly, except when Dynaweb is creating a length report or scanning a large number of data records.

Note that a hotkey does not require an additional BBx operator license.  BBx operator licenses typically have a one time cost between $100 and $175 depending on the number of operators currently licensed.

SMS operators are Dynamo System Management operators.  One Dynamo operator license is required for each logged in operator.  The one time cost for a Dynamo operator license is $100 for the System Management Software, plus $400 for the Dynamo application.  The Excellware Priority Support Plan fees are based on the number of Dynamo operators.

You can use the I for Information option at any menu to display the number of operator licenses currently in use.  Contact Excellware for more information or to purchase additional operator licenses.
