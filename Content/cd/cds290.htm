System Management Configuration

Printer Preference Maintenance

 Last Revised: 02/23/16

This program can be used to limit the number, and control the order, that printers are presented during printer selection. This program can also be used to establish a default or preferred printer.  Each operator can have their own preferences, including the default printer and the group of printers to appear in the initial list of printers.

Use the Printer Group Maintenance option to create a list of printers.  This list is assigned a Group Code. During printer selection, the operator will be presented the list of printers in the same sequence as defined in this program. You may want to create a Printer Group for each department or group of operators that would typically print to the same printers. Once a Printer Groups has been defined, it can be share among multiple users.

Use the other options to assign a Printer Group to a Terminal ID (alias), an IP Address, a Login ID, or PC Name, or the first 3 octet's of the client IP Address.

The Printer Selection process uses the following hierarchy to determine the Default Printer and Printer Group.

  1. The client device full IP Address
  2. The client PC Name or workstation hostname
  3. The Terminal ID (alias)
  4. The User's Login ID
  5. The first 3 octet's of the client device IP address, with .000 as the last octet
  6. A wildcard PC Name/workstation hostname, i.e., one "*" or all "*"s.
  7. For Dynastart connections, if no preference has been identified, then the Printer Group will be set to "DYNS".

Note that the Terminal ID (alias) is typically assigned sequentially for networked devices. Use the Terminal ID option only for legacy hardwired terminals connected via serial cable.

You may also assign a default printer whether or not you assign an IP Address, Terminal, or Login ID to a Printer Group. Similarly, you can assign a Printer Group and not specify a default printer.

During printer selection, the operator may print to a printer not included in the Printer Group, by touching the F5 key to select from the complete list of all printers.

If there is no Printer Group established for a given terminal, then all printers will be listed in order based on the Printer ID established in Printer Control Maintenance.
