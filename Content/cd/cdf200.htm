Data Dictionary Field Rename

 Last Revised: 05/03/07

This program should be used whenever you want to change a Data Dictionary field name for a data file with existing records.

If you change a field name within the Data Dictionary and then use the Compare to Data Dictionary option of the File Utility to convert the data file to match the data dictionary, the conversion program will not know that the old field name and the new field name are related.  The old field will be removed from the file, and the new field will be added with blank or zero values.

To prevent this from happening, always use this program when renaming a field.  It will change both the data dictionary and the template file.

To install the project on the customer's or production server, run this program on that server before installing any changes to the data dictionary.
