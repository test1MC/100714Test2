System Management Utility Functions

Reset System Files

 Last Revised: 05/23/14

This program provides updating to the system files and may be needed to resolve data corruption problems with certain system and dictionary files.

The Reset System Files Screen
Reset System Files
Reset Documentation Control Numbers This option rebuilds the control number in relationship to the main selector. For example, if Sales Order Processing is the first choice off of the main selector, that will be the first number in the upper left screen.
Reset Files Displayed using menu "F" option If the operator types in 'F' it displays files that are open as specified in the File Data Dictionary. It then groups similar files.
Reset/Resize Variable Data Dictionary Files This option is used only if the file name is corrupted. It rebuilds the file and compresses records to downsize the file.
Reset/Resize File Data Dictionary Files This option is used only if the file name is corrupted. It rebuilds the file and compresses records to downsize the file.
