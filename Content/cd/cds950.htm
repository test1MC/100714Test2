Log off Dynamo

 Last Revised: 03/01/16

Select this application to exit the Dynamo Application.

You can also enter E to Exit from any Dynamo menu. If running Dynamo from a PC running a terminal emulator such as FaceTwin, or an Apple Macintosh, do not click the red X to exit. Clicking the red X will terminate the application without allowing it to finish the transaction and without updating the Display System Status information, which shows active users. On Windows PCs, clicking the X will not close Dynamo. A pop-up will tell you that you must exit your application, which can be done by entering an E at any Dynamo menu and pressing Enter.
