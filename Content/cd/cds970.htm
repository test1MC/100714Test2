System Management Utility Functions

Menu Printing

 Last Revised: 02/11/06

This program is used to print the contents of one or more menus.  The operator may choose to print the menu in one of four formats: menu, detail, file cross reference, and multi-task cross reference, depending on the type of information needed. Each format is described below.

The Menu Printing Screen

 2.  Detail Format
 3.  File/Program Cross Reference
 4.  Multi-Tasking Cross Reference

CDS970
Menu Format This format is simply a listing of the menu as it appears on the terminal screen.
Detail Format This format is the most informative. It displays all of the data entered in selector maintenance in a more compact format. It may be used as a detailed reference for the selector system, allowing easy access to information without having to call each line item for each selector on the terminal. The date index, return system code, selector text company, and documentation number are printed first.  Each line number on the menu is printed in the following format:
 
LINE#LINE TITLE PF:(Program File)F5:GHST:
 DOC#:SCRN:F1(Data Files)F6:PASS:
 TYPE:WIDE:F2: F7:ESCP:
 MULT:PAGE:F3: F8:OMIT:
   F4: F9: 
File Cross Reference This option sorts and displays the names of each file used for a given selector or multiple selectors. This format may be used as a quick and easy method of locating a specific file in terms of where it is accessed by the system and the number of different places it is accessed. This is very useful if the format of a file is altered and a change is required to the way the file is read or written to. For each file, the file name, system code, program name, VDT suffix, pass parameter and job description is printed.
Multi-tasking Cross Reference This format displays all items from the selector(s) which have multi- tasking. The information is printed by multi-task position and multi-task type within each position. The system code, program name, VDT suffix, pass parameter, and job description are printed for each job requiring multi-tasking. This format is useful when multi-tasking jobs are being reviewed or when new jobs are being assigned a multi-task number.
