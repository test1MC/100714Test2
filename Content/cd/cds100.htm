System Management Software

General File Maintenance 

Last Revised: 02/11/06

This utility program is used to display, add, modify, or delete records from any file within the system. It is generally only used by programmers to display file information that is not available elsewhere.  Due to the power of this program, it is recommended that this program be run only under the direct guidance of your programmer.  This program was not written by Excellware, and in most cases is only needed when the File Utility (SMS) cannot be used.

1. Enter file name desired. Prompt appears at top left of screen.

Example : FILE MMYD10

The file must be a valid direct direct, sort, or indexed file. Hit 'CR' to retrieve file information.

2. General information about file appears at top of screen.

Example : FILE MMYD10 DISC 0/0 DIRECT 24 RECORDS USED OF 25

For indexed files the total number of defined records displayed is actually one less than the total number of records in the file as it is assumed that index 0 is used to store the number of records currently in use, plus a possible file identification. to get a different file at this point, hit 'CR' or CTL IV.

3. The following will appear at the bottom of the screen:

FILE,
PRINT,
HARD-COPY,
KILL,
DELETE,
LIST,
END(F,P,H,K,D,L,E)

Entry CodeActionExplanation
FFILEChoose a different file.
PPRINT Allows you to print on the VDT all KEYS to a file or specific KEYS. 'START FROM ___' appears at the bottom of the screen. Enter the key at which to begin printing in the blank provided. **This does not right justify numbers - you must do this yourself.

EXAMPLE:  If you desire the record with a key of 4, you must enter __4 (that is, space-space-4), not 4__ (4-space-space).

HHARD COPYCreates a copy of the keys you specify on paper.  First it asks you for the printer you desire to use.  Next it asks for a starting key.
KKILLDelete entire file
DDELETE Delete a single record or several records. At the bottom of the screen this will appear:

DELETE FROM ___ TO___ OK?(Y/N)

To have the same key appear in the second field, hit 'CR'.

LLIST At the bottom of the screen this will appear:

LIST (OR F,P,H,K,D,Z,E,N,I,S,T,M,A,R)___

At this point enter either the key of the record desired to appear on the screen OR enter one of the letters listed. The function of each letter is listed below.
 

F,P,H,K,DFunction the same as listed above
Z Displays original menu at screen bottom :

FILE,PRINT,HARD-COPY,KILL,DELETE,LIST,END(F,P,H,K,D,L,E)

EEnds program, returns to CTL menu
NDisplays next record on screen
CTL I
CTL II
CTL III		All display next record on screen and allow you to scroll through records in file
CRAlways displays first record in file if a record is already displayed
IDisplays first 20 fields of record (fields 1-20)
SDisplays second 20 fields of record (fields 21-40)
TDisplays third 20 fields of record (fields 41-60)
M Changes fields (entire field or partial field)

Appearing at screen bottom:  'ENTER FIELD # (OR 'CR' TO EXIT)___'

Choose either field number (to change entire field) OR portion of field to be changed.

Example: 3(3,2) will change the third field, starting at the third character, length of two characters.

A Adds a record to the file. Appearing at screen bottom:

'ENTER KEY ___'

Enter key of record to be added to the file.

RChanges or renames key
EENDEnds program, returns to CTL menu
